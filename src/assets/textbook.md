# 英文構造ガイドブック

X / Y / Z フレームワークで読み解く英文の体系

このガイドは、英文構造トレーナー（English Structure Trainer）の理論的土台。アプリの Step 3〜6 と連動し、英文を「役割」と「表現方法」の組み合わせで理解する体系を提供する。

---

## はじめに

### この教科書の使い方

- **第1部 入り口**：学習の前提（必読）
- **第2部 部品を知る**：品詞・かたまり・動詞の活用
- **第3部 骨格を作る**：文型と動詞の情報
- **第4部 3つの主役 X / Y / Z**：本書の核
- **第5部 かたまりの体系**：3×3マトリクス・ネスト
- **第6部 機能語の体系**：冠詞・前置詞・接続詞・関係詞
- **第7部 発展**：句動詞・名詞句構造・ニュアンス・その他構文
- **付録**：全体地図・解読手順・用語集・Step対応表

### 中心原則

> 文を理解する ≠「単語を枠に入れること」  
> 文を理解する＝**「どのかたまりが、どの語にかかっているか（係り受け）を確定すること」**

文法とは「かたまりがどの役割で、どこにかかれるか」のルール集。本書はこのルール集を **X / Y / Z** の3役割で体系化する。

### キーセンテンス

本書の起点となる例文：

```
People who work with me work hard when an issue arises.
```

関係詞節（Y）と副詞節（Z）が骨格にぶら下がる、日本人学習者が崩れやすい構造の典型。本書を読み終えると、この文の構造を即座に読み解けるようになる。

## 第2章　英語と日本語の構造差

> アプリ該当：全Stepで意識する

### この章で扱うこと

英語学習で最も大きな壁は、たいてい「単語がわからない」ことだけではありません。**日本語と英語で、文の組み立て方そのものが違う**ことも、大きな壁になります。

この章では、その構造の違いを整理します。ここを押さえると、後の章やアプリの練習で「なぜ英語はこの語順なのか」が見えやすくなります。

### 一言での違い

まず、全体像だけ先に押さえておきましょう。

> 英語は **核（いちばん大事な言葉）を先に出して、説明を後ろに足す**（head-initial）  
> 日本語は **説明を前に積んで、核を最後に出す**（head-final）  
> つまり、**情報の並べる方向が逆**です。

たとえば「本」について話すとき：

- **日本語**：「私が昨日買った**本**」→ 説明が先、**本**という核が最後
- **英語**：「**the book** that I bought yesterday」→ **the book** という核が先、説明が後

同じ内容でも、**どこから読み始めて、どこで意味が確定するか**が違います。

### 4つの具体的な差

ここからは、その「方向の違い」が実際の英文でどう現れるかを、4つのポイントに分けて見ていきます。

---

**(1) 後置修飾 — 名詞の説明が「前」か「後」か**

日本語で名詞を修飾（説明）するとき、修飾語は**名詞の前**に置きます。英語では、多くの修飾が**名詞の後**に来ます。これを **後置修飾** と呼びます。

| 言語 | 例 | 構造 |
|------|----|----|
| 英語 | the book [that I bought yesterday] | 名詞が先、説明が後 |
| 日本語 | [私が昨日買った] 本 | 説明が先、名詞が後 |

**読み方のイメージ**

- **日本語**：「私が昨日買った … 本」と、**最後の「本」まで読んで初めて「何の話か」が確定**する読み方になりがちです。
- **英語**：「the book …」と、**最初に「本の話だ」と分かる**。そのあと `[that I bought yesterday]`（私が昨日買った）という説明が後ろから付いてきます。

つまり、日本語脳には「最後の名詞まで待つ」癖がつきやすい一方、英語は **「名詞 → 後ろに説明が続く」** という流れを前提に読まないと、途中で意味が崩れてしまいます。

**もう1つ例を見る**

| 日本語 | 英語 |
|--------|------|
| ピアノを弾いている**少女** | a girl **playing the piano** |
| 机の上の**本** | the book **on the desk** |

日本語は `[説明] + 名詞`、英語は `名詞 + [説明]` です。翻訳するときも、英語では **名詞を先に置いて、説明を後ろに回す** 必要があります。

---

**(2) 動詞の位置 — 「誰が・何をした」がいつ分かるか**

英語と日本語では、**動詞（何をしたか）が文のどこに来るか**も大きく違います。

**英語の場合**

文のはじめのほうで、主語と動詞が並びます。

```
I bought ...
↑   ↑
誰が  何をした
```

`I bought` だけでも、すでに **「私が買った」** という骨格（だれが・どうした）が確定します。そのあとに `a book yesterday` などの詳細が続き、**左から順に意味が固まっていきます**。

**日本語の場合**

同じ内容でも、動詞は文の**最後**に来ることが多いです。

```
私が  昨日  本を  買った
↑              ↑
誰が           何をした（ここで初めて確定）
```

最後まで読まないと、「結局何をしたのか（買ったのか、読んだのか、見たのか）」が分かりません。

**読解戦略**

> 英語は **最後まで待たない**。左から順に意味を固めていく。

英文を読むときは、「文末の動詞が来るまで保留する」のではなく、**最初に出てきた主語と動詞で骨格を先に固定**しましょう。そのうえで、後ろに続く修飾や詳細を足していく読み方が、英語には合っています。

---

**(3) 関係詞という明示マーカー — 「ここから説明が始まる」合図**

**(1) の後置修飾** では、英語は名詞の**後ろ**に説明を置きます。その「説明が始まった」ことを示す**合図**が、関係詞です。

**日本語の場合**

修飾語を名詞の前に置きますが、**「ここから説明です」という印は付きません**。

- 「**昨日買った**本」→ 「昨日買った」が説明だと、文脈から判断する

**英語の場合**

名詞の後ろに説明を置くとき、**`who` / `which` / `that`** などの関係詞で **「ここから後ろは、その名詞の説明です」** と明示します。

- `the book **that** I bought yesterday`
- `a man **who** speaks English`
- `the city **where** I live`

`that` や `who` を見たら、「この名詞について、後ろに説明が続く」と読み取れます。

**なぜ大事か**

日本語では「名詞の前＝説明」と決まっているので、印がなくても読めます。英語では説明が**後ろ**に来るうえ、節（小さな文）になることも多いので、**この合図を素早く拾えるかどうか**が、読むスピードや理解の正確さを左右します。関係詞に慣れる練習は、英語読解の近道になります。

---

**(4) 省略しない情報 — 英語は「文脈で分かるだろう」を表に出さない**

日本語は、文脈や状況から分かる情報を**省略しても**自然な文になります。英語は、日本語なら省略しがちな情報も**文の中に必ず表す**ことが多いです。

| 情報 | 英語 | 日本語 |
|------|------|--------|
| 主語 | 必須 | 省略可（「行きます」だけでOK） |
| 数（複数）| dogs（-s 必須）| 犬（単数・複数で形が変わらない）|
| 冠詞 | a / the を選ぶ | 冠詞に相当するものがない |
| 前置詞 | 名詞の前に置く（in the room）| 助詞が名詞の後ろに付く（部屋**に**）|

**具体例**

| 日本語 | 英語で必ず出すもの |
|--------|-------------------|
| （私は）行きます | **I** go |
| 犬がいる | **A** dog / **The** dog（冠詞） |
| 犬が3匹いる | **Three dogs**（複数形 -s） |
| 部屋にいる | **in** the room（前置詞） |

日本語では「誰が」「何匹か」「どの犬か」を、前後の文脈に任せて省略できます。英語は、**日本語が文脈に委ねる情報を、できるだけ文の表面に出す**言語です。そのぶん単語数は増えますが、1文だけ読んでも意味が取りやすくなる、という特徴があります。

---

### この章のまとめ

4つの差を並べると、こうなります。

1. **修飾の位置**：日本語は名詞の前、英語は名詞の後（後置修飾）
2. **動詞の位置**：英語は左から骨格が確定、日本語は文末で確定
3. **関係詞**：英語だけが「説明の始まり」を `who / which / that` で合図する
4. **省略**：英語は主語・数・冠詞・前置詞などを省略しにくい

英語を日本語と同じ「最後まで待つ」読み方で処理すると崩れやすいので、**核を先に拾い、左から意味を積み上げる**——この感覚を、以降の章でも意識してください。

## 第4章　品詞 — 8つの主役と機能語

> アプリ該当：Step 1（前提）

### 8つの品詞

| 品詞 | 英名 | 働き | 例 |
|------|------|------|----|
| 名詞 | Noun | もの・こと・人 | dog, water, love |
| 代名詞 | Pronoun | 名詞の代わり | I, you, it, this |
| 動詞 | Verb | 動作・状態 | run, is, have |
| 形容詞 | Adjective | 名詞を説明 | big, happy, red |
| 副詞 | Adverb | 動詞・形容詞・文を説明 | slowly, very, here |
| 前置詞 | Preposition | 名詞の前で関係を作る | in, on, at, with |
| 接続詞 | Conjunction | 語や文をつなぐ | and, but, because |
| 冠詞 | Article | 名詞の前の標識 | a, the |

### 主役は3つだけ

8つのうち意味を担う **主役は3つ**：

| 記号 | 役割 | 元の品詞 |
|------|------|---------|
| **X** | 名詞役 | 名詞・代名詞 |
| **Y** | 形容詞役 | 形容詞 |
| **Z** | 副詞役 | 副詞 |

残りは **機能語（つなぎ・標識）**：前置詞・接続詞・冠詞・関係詞。

### 例文での確認

> `I usually buy bread and butter at a local supermarket.`

I（代名詞）／ usually（副詞）／ buy（動詞）／ bread・butter（名詞）／ and（接続詞）／ at（前置詞）／ a（冠詞）／ local（形容詞）／ supermarket（名詞）

## 第5章　単語・句・節 — 大きさによる3分類

> アプリ該当：全Stepの前提

### 判定軸は1つ

> 中に「S＋V」のセットがあるか？

| 種類 | 定義 | 例 |
|------|------|----|
| 単語（word）| 1語 | morning |
| 句（phrase）| 2語以上・S+V **なし** | in the morning |
| 節（clause）| 2語以上・S+V **あり** | when I was young |

### 例

- `in the morning` → S+V なし → **句**
- `when I was young` → "I was" がS+V → **節**

### なぜこの軸が重要か

句と節では「文の中での扱い」が変わる。節は内部が小さな文として成立しているため、節の中にもまた X / Y / Z の構造が再帰的に現れる（→ 第15章）。

## 第6章　動詞の5変化

> アプリ該当：Step 3 の前提

### 動詞が取る5つの形

| 形 | 名称 | 例（play） |
|----|------|-----------|
| 原形 | base | play |
| 三単現 | 3rd-person singular | plays |
| 過去形 | past | played |
| 過去分詞 | past participle | played |
| 現在分詞 | -ing form | playing |

### これらが組み合わさって何を作るか

| 形 | 使われる場面 |
|----|------------|
| 原形 | 助動詞の後・命令文・不定詞 |
| 三単現 | 主語が3人称単数の現在形 |
| 過去形 | 過去のことを述べる |
| 過去分詞 | 完了形（have + 過去分詞）・受動態（be + 過去分詞）・分詞の形容詞用法 |
| 現在分詞 | 進行形（be + -ing）・動名詞・分詞の形容詞用法・分詞構文 |

5変化は **時制・相・態・準動詞すべての材料** となる。ここが脆弱だと第9章以降が崩れる。

### 不規則動詞

代表的なものを覚える必要がある：

```
be    – was/were – been
have  – had     – had
do    – did     – done
go    – went    – gone
see   – saw     – seen
take  – took    – taken
write – wrote   – written
read  – read    – read  （発音だけ変わる）
```

## 第7章　5つの文型

> アプリ該当：Step 2（前提）

### 文型一覧

| 文型 | 構造 | 例 |
|------|------|-----|
| 第1 | S + V | He runs. |
| 第2 | S + V + C | He is happy. |
| 第3 | S + V + O | She bought a pen. |
| 第4 | S + V + O + O | He gave her a book. |
| 第5 | S + V + O + C | They elected him president. |

### O と C の見分け方

- **O（目的語）**：主語と「別物」を指す名詞
  - She bought **a pen**. → "she" と "a pen" は別物
- **C（補語）**：主語または目的語を「説明」する語
  - He is **happy**. → "he" の状態を説明
  - They elected him **president**. → "him" の役割を説明

### 補語 C の中身

C には2種類ある（→ 第10・11章で詳述）：
- **形容詞（Y）**：He is happy.
- **名詞（X）**：He became a doctor.

→ 文型公式では `C = X または Y` と表記する。

## 第8章　動詞のタイプと文型の対応

> アプリ該当：Step 2 の補完

### 動詞のタイプは文型を決める

文型を選ぶのは話者ではなく **動詞そのもの**。各動詞は取れる文型が決まっている。

| 動詞のタイプ | 取れる文型 | 代表例 |
|------------|-----------|--------|
| 完全自動詞 | SV | run / sleep / arrive |
| 不完全自動詞（連結動詞）| SVC | be / become / seem / look / feel |
| 完全他動詞 | SVO | eat / read / want |
| 授与動詞 | SVOO | give / tell / show / send |
| 不完全他動詞 | SVOC | make / find / call / elect |
| 知覚動詞 | SVO + 原形/-ing/-ed | see / hear / feel / watch |
| 使役動詞 | SVO + 原形/to do/-ed | make / have / let / get |

### 動詞起点で読む

英文を読むときは **まず動詞を見つける**。動詞が決まると、その後ろに何が来るかが予測できる。

> 例：`become` を見たら → 後ろに C（補語）が来ると予測 → "a doctor" は X 役の C と判定

## 第9章　動詞が背負う4情報

> アプリ該当：Step 3 本体

### 動詞は4つの情報を背負う

| 情報 | 軸数 | 内容 |
|------|------|------|
| 時制（Tense）| 3 | 現在 / 過去 / 未来（will） |
| 相（Aspect）| 4 | 単純 / 進行（be+-ing）/ 完了（have+pp）/ 完了進行（have been+-ing）|
| 態（Voice）| 2 | 能動 / 受動（be+pp）|
| 法（Modality）| n | 直説法 / 仮定法 / 助動詞（can/must/may/will/should...）|

### 時制 × 相 = 12形

| | 単純 | 進行 | 完了 | 完了進行 |
|--|------|------|------|---------|
| 現在 | plays | is playing | has played | has been playing |
| 過去 | played | was playing | had played | had been playing |
| 未来 | will play | will be playing | will have played | will have been playing |

### 態を加えると24形

これに受動態（be + 過去分詞）が加わる。さらに助動詞・否定が組み合わさる。

### 何を訓練するか

Step 3 では、日本語の「いつ・どんな様子で・する/される・どのくらい確実か」を、上記マトリクスのどの形に対応させるかを練習する。

## 第10章　X — 名詞役

> アプリ該当：全Stepで使用、Step 5・6 で集中訓練

### X の働き

S（主語）・O（目的語）・C（補語の一部）になれる。

### X が取る形

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 名詞・代名詞 | book / I |
| 句 | 名詞句 | a business book |
| 句 | 動名詞句（-ing）| Playing tennis is fun. |
| 句 | 不定詞句（to do）| To play tennis is fun. |
| 句 | 疑問詞+不定詞 | I don't know what to do. |
| 節 | that節 | I know that he is honest. |
| 節 | whether/if節 | I wonder whether it is true. |
| 節 | wh節（疑問詞節）| I know what you said. |

### 節の3パターンの理由

| 節 | 元になる文 |
|----|-----------|
| that節 | 平叙文 → He is honest. |
| whether/if節 | Yes/No疑問文 → Is it true? |
| wh節 | 情報疑問文 → What did you say? |

3つで名詞節を **MECE に網羅** する。

## 第11章　Y — 形容詞役と修飾の方向

> アプリ該当：Step 4・5 で集中訓練

### Y の働き

名詞を **修飾する**（飾る・説明する）。

### Y が取る形

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 形容詞 | an **interesting** book |
| 単語 | 分詞単独 | a **broken** window / a **sleeping** baby |
| 句 | 前置詞句 | the book **on the desk** |
| 句 | 不定詞句 | a book **to read** |
| 句 | 現在分詞句 | a girl **playing the piano** |
| 句 | 過去分詞句 | a book **written by her** |
| 句 | 同格 | my friend **Tom** |
| 句 | 形容詞句 | a **very tall** man |
| 節 | 関係代名詞節 | the book **that I bought** |
| 節 | 関係副詞節 | the city **where I live** |
| 節 | 同格節 that | the fact **that he lied** |

### 修飾の方向

英語特有のルール：

| 修飾するもの | 配置 | 例 |
|------------|------|-----|
| 単語（形容詞・分詞単独）| **前置** | a **red** book |
| 句（前置詞句・分詞句・不定詞句など）| **後置** | the book **on the desk** |
| 節（関係詞節・同格節）| **後置** | the book **that I bought** |

**例外**：不定代名詞（something / anything / nobody）は単語でも後置  
→ `something **interesting**`（× interesting something）

### 関係詞節と同格節の違い

両方とも `名詞 + that + S + V` の形だが、構造が違う：

- **関係詞節（gap あり）**：the book [that I read ___]  
  → that が clause 内の空所（"I read [the book]"）を埋める
- **同格節 that（gap なし）**：the fact [that he left]  
  → clause がそのまま名詞の内容を説明する（he left がそのまま完成形）

---

## 第12章　Z — 副詞役と配置ルール

> アプリ該当：Step 4・6 で集中訓練

### Z の働き

動詞・形容詞・他の副詞・文全体を **修飾する**。

### Z が取る形

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 副詞 | walked **slowly** |
| 句 | 前置詞句 | studied **in the library** |
| 句 | 不定詞句 | went there **to see her** |
| 句 | 分詞構文 | **Walking home**, he met her. |
| 句 | 副詞句 | walked **very slowly** |
| 句 | 名詞句の副詞用法 | gets up at 6 **every morning** |
| 節 | 副詞節 | left **when it rained** |

### Z の3つの配置位置

```
① 前置（Front）：文頭・主語の前
② 中置（Mid）   ：主語直後・動詞直前（be / 助動詞の後）
③ 後置（End）   ：動詞句の後ろ（文末）
```

### 種類別の好み

| 種類 | 好みの位置 | 例 |
|------|-----------|-----|
| 様態 | ③ 後置（または②）| She walked home **slowly**. |
| 頻度 | ② 中置（強い）| She **always** goes home early. |
| 時 | ③ 後置 または ① 前置 | She went home **yesterday**. |
| 場所 | ③ 後置 | She waited **outside**. |
| 程度 | 修飾する語の直前 | She walked **very** slowly. |
| 文修飾 | ① 前置 または ② 中置 | **Unfortunately**, she left. |

### 重要なルール

**(1) 目的語を飛び越えない**

```
○ She read the book slowly.   ← 目的語の後ろ
× She read slowly the book.   ← 非自然
```

**(2) 複数の副詞が並ぶときの順**

> 様態 → 場所 → 時

```
She walked slowly (様態) home (場所) yesterday (時).
```

### 副詞節の細分

副詞節は接続詞で意味が決まる：

| 種類 | 接続詞 |
|------|--------|
| 時 | when / while / before / after / until / since |
| 理由 | because / since / as / now that |
| 条件 | if / unless / in case |
| 譲歩 | although / though / even though |
| 目的 | so that / in order that |
| 結果 | so ... that / such ... that |
| 様態 | as / as if / as though |
| 比較 | than / as ... as |
| 場所 | where / wherever |

## 第13章　3×3マトリクス — 役割 × 表現方法の全体像

> アプリ該当：本書の核

### マトリクス本体

|  | Word（単語）| Phrase（句・S+Vなし）| Clause（節・S+Vあり）|
|--|------------|--------------------|---------------------|
| **X** 名詞役 | 名詞・代名詞 | 名詞句・動名詞句・不定詞句・疑問詞+不定詞 | that節・whether/if節・wh節 |
| **Y** 形容詞役 | 形容詞・分詞単独 | 前置詞句・不定詞句・分詞句・同格・形容詞句 | 関係代名詞節・関係副詞節・同格節 that |
| **Z** 副詞役 | 副詞 | 前置詞句・不定詞句・分詞構文・副詞句・名詞句の副詞用法 | 副詞節（時/理由/条件/譲歩/目的/結果/様態/比較/場所）|

### 高タイパなパーツ

複数のセルに登場する形は **学習コスパが高い**：

| 形 | 出現するセル | 効果 |
|----|------------|------|
| 不定詞（to do）| X・Y・Z 全て | 1つの形で3役割をカバー |
| 前置詞句 | Y・Z 両方 | 1つの形で2役割 |
| -ing形 | X（動名詞）・Y（分詞）・Z（分詞構文）| 1つの形で3役割 |
| 分詞 | Y（後置修飾）・Z（分詞構文）| 1つの形で2役割 |

→ Step 4 でこれらの **準動詞（不定詞・動名詞・分詞）と前置詞句** を集中訓練するのは、ここに高タイパが集中しているから。

## 第14章　同じ形が複数の役割を持つ

> アプリ該当：全Stepで意識

### MECE にならない理由

第13章のマトリクスは「形」では MECE にならない。同じ形が複数の役割を持つことが頻繁にある。

### 主要な「同形・複数役割」一覧

| 形 | X | Y | Z |
|----|---|---|---|
| 不定詞 to do | ○ | ○ | ○ |
| 前置詞句 | - | ○ | ○ |
| -ing | ○（動名詞）| ○（分詞）| ○（分詞構文）|
| that節 | ○（名詞節）| ○（同格節・関係詞節）| - |
| when / where 等 | ○（wh節）| ○（関係副詞節）| ○（副詞節）|

### 具体例で違いを見る

**不定詞 to do**

| 役割 | 例 |
|------|-----|
| X | **To read** is fun.（主語）|
| Y | a book **to read**（books を後置修飾）|
| Z | I went there **to read**.（目的・副詞役）|

**when 節**

| 役割 | 例 |
|------|-----|
| X | I know **when he left**.（名詞節・目的語）|
| Y | the day **when he left**（day を後置修飾）|
| Z | He left **when it rained**.（副詞節・時）|

### 結論

> **形ではなく、文中での位置と働きで役割が決まる**

これが英文解読の核心。形を覚えるだけでは不十分で、各かたまりの **役割** を判定する訓練が必要。

## 第15章　ネストの再帰性

> アプリ該当：Step 5・6 で集中訓練

### 中心原則

> X / Y / Z のルールは句・節の **内部でも再帰的に適用される**

つまり、節の中身もまた一つの小さな文として X / Y / Z で解析できる。

### 例1：節の中のネスト

```
I know [that she slowly reads books at night].
         └──── 名詞節（X）────┘
              ├─ she（X・主語）
              ├─ slowly（Z・様態副詞・中置）
              ├─ reads books（V + X）
              └─ at night（Z・時の前置詞句・後置）
```

### 例2：複数のかたまりのネスト

```
People [who work with me] work hard [when an issue arises].
        └──── Y（関係詞節）─┘         └──── Z（副詞節）────┘
```

骨格：People work hard  
→ Y（who節）が "People" を後置修飾  
→ Z（when節）が "work hard" を修飾  
→ when節の内部にも an issue (X) + arises (V) がネスト

### 句の中のネスト

```
the book [on the desk]
          └─ 前置詞句（Y）
              └─ "the desk" = X（名詞）が句の中にネスト
```

### 読解上の意味

ネストは「処理できるかたまりの長さと深さ」が学習進度を決める。  
Step が進むほど、ネストの深さも増す。アプリの Step 6 はこのネストを正面から扱う。

# 第6部　機能語の体系

## 第16章　冠詞・限定詞

> アプリ該当：Step 4 以降の名詞句構成

### 名詞の前の標識

冠詞・限定詞は単独で文中の主役にはならないが、名詞句を構成する必須の部品。

| 種類 | 例 |
|------|-----|
| 冠詞 | a / an / the |
| 指示詞 | this / that / these / those |
| 所有格 | my / your / his / her / its / our / their |
| 数量詞 | some / any / many / much / few / several / all |
| 数詞 | one / two / three ... |

### 名詞句の組み立て順

```
[限定詞] [数詞] [形容詞] 名詞 [後置修飾]
   the      two       tall      men     in the park
```

→ 一つの名詞句の中に **限定詞・形容詞（Y/Word）・名詞（X/Word の核）・修飾句（Y/Phrase）** がネストしている。

### a / the の使い分けの原理

- **a / an**：話し手と聞き手の共通認識にない、初出または不特定の名詞
- **the**：話し手と聞き手の共通認識にある特定の名詞

詳細は別途学習が必要だが、X 役の名詞句を作る上で必須。

---

## 第17章　前置詞

> アプリ該当：Step 4 で集中訓練

### 前置詞の役割

前置詞は **名詞を Y または Z に変換する** 装置。

```
in + the park（名詞・X）→ "in the park" = Y または Z
```

### 主な前置詞の分類

| 種類 | 前置詞 |
|------|--------|
| 位置・場所 | in / on / at / under / between / among / behind |
| 時 | at / on / in / before / after / during / since / for |
| 方向 | to / from / into / out of / toward / through |
| 関係 | with / by / for / about / against |

### 前置詞句は Y か Z か

文の中での位置で決まる：

- **名詞の直後** → Y（その名詞を後置修飾）  
  the book **on the desk** → "on the desk" が "book" を修飾
- **動詞の周辺** → Z（その動詞を修飾）  
  I study **in the library** → "in the library" が "study" を修飾

### 副詞として機能する単語との対比

`home / here / there / abroad / upstairs / inside / outside / away / back` などは **それ自体が副詞** → 前置詞不要：

```
○ go home / × go to home
○ come here / × come to here
```

ただし所有格を付けて名詞扱いなら前置詞必要：

```
○ go to my home（"home" を名詞として使う）
```

## 第18章　接続詞

> アプリ該当：Step 6 で集中訓練

### 接続詞は4種類

| 種類 | 役割 | 代表例 |
|------|------|--------|
| 等位接続詞 | 対等な要素をつなぐ | for / and / nor / but / or / yet / so |
| 従属接続詞（副詞節用）| 副詞節を導入 | when / because / if / although / while |
| 名詞節導入語 | 名詞節を導入 | that / whether / if |
| 相関接続詞 | 相関的に対をなす | both...and / either...or / neither...nor / not only...but also |

### 等位接続詞 FANBOYS

**F**or（理由）/ **A**nd / **N**or / **B**ut / **O**r / **Y**et（しかし）/ **S**o（だから）

頭文字で覚える。同じ品詞・同じ機能のものをつなぐ：

```
I bought bread and butter.（名詞 and 名詞）
She studied hard but failed.（動詞 but 動詞）
```

### 従属接続詞 + 節 = Z（副詞節）

詳細は第12章を参照。意味カテゴリで MECE に分かれる：時・理由・条件・譲歩・目的・結果・様態・比較・場所。

## 第19章　関係詞

> アプリ該当：Step 5 で集中訓練

### 関係詞は2系統

| 種類 | 形 | 役割 |
|------|----|----|
| 関係代名詞 | who / whom / whose / which / that / what | 名詞の代わりに節に入り、節全体を Y にする（what だけ X）|
| 関係副詞 | where / when / why | 副詞の代わりに節に入り、節全体を Y にする |

### 関係代名詞の使い分け

| 先行詞 | 主格 | 目的格 | 所有格 |
|--------|------|--------|--------|
| 人 | who / that | whom / who / that / 省略 | whose |
| 物・事 | which / that | which / that / 省略 | whose / of which |

### 関係副詞の使い分け

| 先行詞 | 関係副詞 | 例 |
|--------|---------|----|
| 場所 | where | the city **where I live** |
| 時 | when | the day **when we met** |
| 理由 | why | the reason **why he left** |

### 関係代名詞 what の特別性

what は **先行詞を含む** 関係代名詞。「〜するもの・こと」の意味：

```
What he said is true.   = The thing that he said is true.
                          ↑ X（名詞節）
```

→ what 節は Y ではなく **X**（名詞節）として機能する。

### 関係詞と疑問文の連続性

関係詞節と疑問文は同じ「空所を埋める」発想で作られる：

| 用途 | 空所の埋め方 |
|------|------------|
| 疑問文 | 空所を **文頭に出して「聞く」** |
| 関係詞節 | 空所を **文頭に出して「名詞を説明する」** |

```
疑問文：       What did you buy?         （空所＝何）
関係詞節：    the thing that you bought  （空所＝何）
```

## 第20章　句動詞

### 句動詞とは

複数の語が一体となって1つの動詞として機能するもの。V のかたまりは1語とは限らない。

### 種類

| 種類 | 例 |
|------|-----|
| 動詞 + 前置詞 | look at / listen to / depend on |

```
She looked at the picture.
    └────V────┘  └──X(目的語)──┘
```

## 第22章　ニュアンスと表現の選択

> アプリ該当：全Stepで意識

### 同じ日本語でも英語の構造は1対1でない

```
「彼女には読むべき本がたくさんある」
```

これは少なくとも以下3通りに訳せる：

| 英訳 | ニュアンス |
|------|-----------|
| She has many books **to read**. | 積読・読む予定（量の話）|
| She has many books **she should read**. | 外部起点の推奨・義務 |
| She has many books **worth reading**. | 読む価値がある（価値判断）|

### 不定詞（to do）の Y 用法のニュアンス

**「これから／予定／可能」**を表す。義務や推奨ではない：

```
a book to read       = これから読む本
something to eat     = 食べるもの（これから食べる）
a place to visit     = 訪れる場所（予定）
```

### 義務・推奨を表したいとき

| 表現 | 構造 |
|------|------|
| books she should read | 関係詞節（Y/Clause）|
| books worth reading | 形容詞句（Y/Phrase）|
| recommended books | 過去分詞（Y/Word）|

### 学習者の意識

英作文では「自分の日本語のニュアンスがどれか」を見極めてから構造を選ぶ。同じ日本語に複数の英訳がありうることを前提に置く。

|------|----|----|

| 比較級 | -er than / more ... than | taller than me |

# 付録

## 付録 A. 全体地図

```
英文の理解
│
├── ① 骨格（文型 ＝ 動詞のタイプで決まる）
│      ├── SV / SVC / SVO / SVOO / SVOC
│      └── 動詞は4情報を背負う：時制(3) × 相(4) × 態(2) × 法
│
├── ② かたまり（X / Y / Z の3役割）
│      ├── X 名詞役：単語 / 句 / 節 → S・O・C を埋める
│      ├── Y 形容詞役：単語(前置) / 句・節(後置) → 名詞を飾る
│      └── Z 副詞役：単語 / 句 / 節 → 動詞・文を飾る
│
├── ③ 接続（骨格が複数のとき）
│      ├── 等位接続：and / but / or ...
│      └── 従属接続：副詞節・名詞節・関係詞節として接続
│
└── ④ 機能語（X / Y / Z と並ぶ"つなぎ"の体系）
       ├── 冠詞・限定詞：名詞の前の標識
       ├── 前置詞：名詞を Y / Z に変換
       ├── 接続詞：節と節をつなぐ
       └── 関係詞：節を Y / X に変換

→ 全体に対して、以下は "その他"（後回し可）：
   - 比較・仮定法・倒置・強調構文・特殊構文（there / 命令 / 感嘆）
```

---

## 付録 B. 文を解く順序（実践パーシング）

長い英文に遭遇したときの実践手順。

### Step 1：骨格を見つける

1. **動詞（V）を見つける**：時制が付いている動詞が骨格の動詞
2. **その左の主語（S）を確定する**：動詞の前にある最も近い名詞要素
3. **動詞のタイプから O / C の有無を予測する**（→ 第8章）

### Step 2：各位置のかたまりを確定する

S・O・C それぞれが：
- 単語の名詞か？
- 句か（動名詞句・不定詞句・名詞句）？
- 節か（that節・wh節・whether節）？

### Step 3：修飾要素（Y / Z）を切り分ける

骨格に含まれない部分は Y か Z：
- 名詞の直後にあるものは Y（後置修飾）と仮定
- 動詞の周辺にあるものは Z と仮定
- 文頭・文末の独立した句・節は Z の可能性が高い

### Step 4：かたまりの中にネストがあれば再帰的に同じ手順を適用

節の中身も同じ手順で骨格を見つける（→ 第15章）。

### 例題

```
The student who studies hard usually gets good grades.
```

1. 動詞を探す：`studies` と `gets` の2つ。time-marked は `studies`（現在形）と `gets`（現在形）。  
   骨格の動詞は文末側の `gets`。`studies` は内部の節の動詞。
2. 主語を確定：`The student` ＋ 後ろの関係詞節 = X
3. 動詞 gets の後ろ：`good grades` = X（目的語）
4. 修飾要素：`who studies hard` = Y（関係詞節、student を修飾）/ `usually` = Z（中置・頻度副詞）

最終構造：

```
[The student who studies hard] usually gets [good grades].
        X（主語・名詞句 + Y）       Z       V       X（目的語）
```

## 付録 C. 用語集

| 用語 | 定義 |
|------|------|
| X | 名詞役。S・O・C を埋めるかたまり |
| Y | 形容詞役。名詞を修飾するかたまり |
| Z | 副詞役。動詞・形容詞・文を修飾するかたまり |
| 単語 | 1語のかたまり |
| 句 | 2語以上で S+V を含まないかたまり |
| 節 | 2語以上で S+V を含むかたまり |
| 骨格 | 文型（S + V + O / C）の構造 |
| 文型 | 5つの基本構造（SV / SVC / SVO / SVOO / SVOC）|
| 時制 | 現在 / 過去 / 未来 |
| 相 | 単純 / 進行 / 完了 / 完了進行 |
| 態 | 能動 / 受動 |
| 法 | 直説法 / 仮定法 / 助動詞による表現 |
| 準動詞 | 不定詞・動名詞・分詞 |
| 後置修飾 | 名詞の後ろから修飾すること（句・節がY役を取るとき）|
| 前置修飾 | 名詞の前から修飾すること（単語のYを取るとき）|
| 関係詞 | 節を Y / X に変換する語（関係代名詞・関係副詞）|
| 機能語 | X / Y / Z の主役以外の、つなぎ・標識の語（冠詞・前置詞・接続詞・関係詞）|
| ネスト | あるかたまりの内部に、別のかたまりが入っている構造 |

## 付録 E. よくある誤り

| 誤り | 原因 | 正しい形 |
|------|------|---------|
| go to home | home が副詞であることを失念 | go home |
| She read slowly the book. | 様態副詞が目的語を飛び越えた | She read the book slowly. |
| What who did he give? | 1文に疑問詞を2つ使った | What did he give her? |
| I have a pen what you borrowed. | what は先行詞を含むため、前に名詞があると非文法 | I have a pen that you borrowed. |
| She have many books. | 三単現の -s を忘れた | She has many books. |
| The book on the desk are mine. | 主語の核（book = 単数）と動詞の数を一致させ忘れた | The book on the desk is mine. |

## 結びに

本書は **「英文を X / Y / Z の3役割で読み解く」** という1つのフレームを軸に、英文構造の全体を体系化したガイドである。各章はアプリの該当 Step と紐づいており、アプリで詰まったときの参照書として、また学習計画を立てる地図として活用できる。

最後に、本書の中心原則を繰り返す：

> 文を理解する＝**「どのかたまりが、どの語にかかっているか（係り受け）を確定すること」**

各かたまりについて「**X / Y / Z** のどれか」と「**何にかかっているか**」を確定する作業を、扱えるかたまりを伸ばしながら反復する。これが英文理解の積み上げそのものである。

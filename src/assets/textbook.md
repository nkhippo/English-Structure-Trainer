# 英文構造ガイドブック

X / Y / Z フレームワークで読み解く英文の体系

このガイドは、英文構造トレーナー（English Structure Trainer）の**理論の土台**です。アプリの Step 3〜6 と連動しており、英文を **「役割」** と **「表現方法」** の組み合わせで理解するための体系を提供します。

「単語の意味は分かるのに、文全体の意味が取れない」「英作文で語順が決まらない」——そういったときに、このガイドが地図の役割を果たします。

## はじめに

### この教科書の使い方

本書は7つの部と付録で構成されています。**最初から順番に読む**のがおすすめですが、詰まった章だけ参照しても構いません。

- [第1部 入り口](#part-1)：学習の前提（必読）
- [第2部 部品を知る](#part-2)：品詞・かたまり・動詞の活用
- [第3部 骨格を作る](#part-3)：文型と動詞の情報
- [第4部 3つの主役 X / Y / Z](#part-4)：本書の核
- [第5部 かたまりの体系](#part-5)：3×3マトリクス・ネスト
- [第6部 機能語の体系](#part-6)：冠詞・前置詞・接続詞・関係詞
- [第7部 発展](#part-7)：句動詞・名詞句構造・ニュアンス・その他構文
- [付録](#appendix)：全体地図・解読手順・用語集・Step対応表

アプリで練習しながら、該当する章を開いて「なぜこの語順なのか」を確認する使い方も有効です。

### 中心原則

英文を理解するとき、多くの人は「単語を品詞ごとに分類して枠に入れる」ことを目標にしがちです。しかし本書が目指すのは、それとは少し違います。

> 文を理解する ≠「単語を枠に入れること」  
> 文を理解する＝**「どのかたまりが、どの語にかかっているか（係り受け）を確定すること」**

#### 「枠に入れる」と「係り受けを確定する」の違い

**枠に入れる** = 各単語に品詞ラベルを貼る作業  
**係り受けを確定する** = 「このかたまりは、文の中で**何の説明・修飾になっているか**」を決める作業

同じ文でも、やることが違います。

| やり方 | 結果 | 足りないこと |
|--------|------|-------------|
| 枠に入れる | slowly＝副詞、book＝名詞、read＝動詞 | 意味のつながりが分からない |
| 係り受けを確定 | slowly → read にかかる | 文全体の意味が再構成できる |

#### 例1：`slowly` — ラベルだけでは足りない

`She read the book slowly.`

| 単語 | 品詞ラベル | 係り受け（本書が求めるもの） |
|------|-----------|---------------------------|
| She | 代名詞 | 主語（文の主役） |
| read | 動詞 | 骨格の動詞 |
| the book | 名詞句 | read の目的語 |
| slowly | 副詞 | **read にかかる**（ゆっくり読んだ） |

`slowly` を「副詞」とラベル付けするだけでは不十分です。大事なのは **「slowly が walk（read）にかかっているのか、文全体にかかっているのか」** を確定することです。

この文では `slowly` は **read** にかかります。「本がゆっくり」ではなく「ゆっくり読んだ」です。

#### 例2：同じ品詞でも、かかる先が違う

`very` も副詞ですが、**かかる先**によって意味が変わります。

| 文 | very の係り受け | 意味 |
|----|----------------|------|
| She walked **very** slowly. | slowly にかかる | とてもゆっくり歩いた |
| It was **very** interesting. | interesting にかかる | とてもおもしろかった |
| **Very** interesting books sell well. | interesting にかかる | とてもおもしろい本は… |

品詞が同じ「副詞」でも、**何にかかっているか** が違えば、文の意味も変わります。

#### 例3：かたまり単位で見る

英文は、1語ずつではなく **かたまり（chunk）** 単位で理解する方が早いです。

`The student who studies hard usually gets good grades.`

| かたまり | 役割 | 何にかかっているか |
|---------|------|------------------|
| The student | X（主語） | 文の主役 |
| who studies hard | Y（関係詞節） | **The student** を説明 |
| usually | Z（副詞） | **gets** を修飾（いつも） |
| gets | V（動詞） | 骨格 |
| good grades | X（目的語） | **gets** の目的語 |

「who studies hard」を1語ずつ分解するより、**The student を修飾しているかたまり** としてまとめて捉える方が、意味が取りやすくなります。

#### 文法とは何か — X / Y / Z への接続

文法とは、**「かたまりがどの役割で、どこにかかれるか」** のルール集です。本書はこのルール集を **X / Y / Z** の3役割で体系化します。

| 記号 | 役割 | 典型の係り受け |
|------|------|--------------|
| **X** | 名詞役 | 主語・目的語・補語になる |
| **Y** | 形容詞役 | **名詞** にかかる（名詞を修飾・説明） |
| **Z** | 副詞役 | **動詞・形容詞・文** にかかる |

係り受けを確定する作業は、最終的に **「このかたまりは X / Y / Z のどれか」** と **「何にかかっているか」** の2点に帰着します（→ [第1章](#ch-1)）。

**なぜ品詞名ではなく X / Y / Z なのか** — X / Y / Z は本書独自の造語です。品詞との違い、3つに簡略化する理由、X・Y・Z それぞれの意義は [第4章](#ch-4) で詳しく説明します。品詞の知識はそのまま活かしつつ、本書では **文の中での働き（役割）** を軸に読み進めます。

### キーセンテンス

本書の起点となる例文：

```
People who work with me work hard when an issue arises.
```

（意味：一緒に働く人々は、問題が起きたときに一生懸命働く。）

この文は、**関係詞節（Y）** と **副詞節（Z）** が骨格にぶら下がる、日本人学習者が崩れやすい構造の典型です。以下、段階的に読み解きます。

#### Step 1：骨格だけ先に取り出す

まず修飾を無視して、**だれが・何をした** だけを拾います。

```
People ... work hard ...
↑              ↑
主語           動詞
```

骨格は **`People work hard`**（人々は一生懸命働く）です。

**ここで押さえること**

- 英語は左から骨格が確定する（第2章）
- `People` を見た時点で「人々の話」と分かる
- `work hard` で「一生懸命働く」と確定する
- この時点で、文の**中心の意味**はすでに取れている

#### Step 2：People の説明を後ろから足す（Y）

```
People [who work with me] work hard ...
        └──── 関係詞節 ────┘
```

- `who work with me` = **Y（関係詞節）**
- **People** にかかる（「どんな人々か」を説明）
- 日本語：「**私と一緒に働く**人々」

`who` は「ここから People の説明が始まる」合図です（第2章・第19章）。People の**直後**に来ているので、People を修飾する Y だと判断できます。

#### Step 3：work hard の説明を足す（Z）

```
People who work with me work hard [when an issue arises].
                                    └──── 副詞節 ────┘
```

- `when an issue arises` = **Z（副詞節）**
- **work hard** にかかる（「いつ一生懸命働くか」を説明）
- 日本語：「**問題が起きたとき**に一生懸命働く」

`when` は副詞節の始まりを示す接続詞です。動詞 `work hard` の**周辺**（この文では文末）にあるので Z です。

#### Step 4：節の中も小さな文として見る（ネスト）

`when an issue arises` の中にも、骨格があります。

```
when [an issue] [arises]
      ↑ X        ↑ V
```

- `an issue` = X（主語）
- `arises` = V（動詞）
- 意味：「問題が起きる」

節の中身も **X / Y / Z** で分解できます（→ 第15章）。大きな文は、この「節の中にまた骨格」が入れ子になっています。

#### 全体構造のまとめ

```
People [who work with me] work hard [when an issue arises].
  X         Y（People にかかる）   V    Z（work hard にかかる）
                                      └─ 内部：an issue(X) + arises(V)
```

| かたまり | 役割 | 係り受け |
|---------|------|---------|
| People | X（主語） | 文の主役 |
| who work with me | Y（関係詞節） | People を修飾 |
| work hard | V（動詞） | 骨格 |
| when an issue arises | Z（副詞節） | work hard を修飾 |

#### なぜこの文が「キー」なのか

1. **骨格**（People work hard）が短く、先に固定しやすい
2. **Y**（名詞の後置修飾）と **Z**（動詞の修飾）が**1文に共存**している
3. **Z の節の中**にもネスト（an issue arises）がある

Step 5（関係詞）・Step 6（節とネスト）で扱う構造が、すべてこの1文に含まれています。

本書を読み終えると、この文の構造を**即座に読み解ける**ようになることを目指しています。途中でつまずいても、最後まで戻ってこの例文を見直すと、全体像がつかみやすくなります。

## 第1章　英語を理解するとは

> アプリ該当：全Step共通の前提

### この章で扱うこと

英語の文を、なぜ **「役割と係り受け」** の視点で見るのか——その動機を示します。

単語の意味を1語ずつ調べるだけでは、長い文や入れ子の多い文でつまずきます。本書では、**文の中で各かたまりがどんな役割を担い、何にかかっているか** を確定する方法を学びます。

### 理解の3階層

英文を理解する作業は、大きく3つのレイヤーに分けられます。

1. **単語を理解する** — 各単語の意味を知る
2. **文を理解する** ← **本書のフォーカス**
3. **ストーリー（論理）を理解する** — 複数の文がつながった議論・物語全体を把握する

**具体例**

| レイヤー | 問い | 例 |
|---------|------|-----|
| 単語 | `book` は何？ | 本 |
| 文 | 誰が・何をした？ `slowly` は何にかかる？ | She read the book slowly. |
| ストーリー | なぜその本を読んだのか？ | 前後の段落から推測 |

**文の理解**は、単語と論理の**橋**です。ここを体系化することで、上下のレイヤーも安定します。単語力があっても文の構造が読めなければ論理まで届きませんし、逆に文の構造が読めれば、知らない単語があっても文脈から意味を推測しやすくなります。

### なぜ品詞名ではなく X / Y / Z なのか

学校文法では「名詞・動詞・形容詞…」と**品詞**で分類します。本書が使う **X / Y / Z** は、文法用語として広く定着している言葉ではなく、**本書独自の造語**です。品詞の別名でも、英語の Noun / Adjective / Adverb の略称でもありません。

X / Y / Z は **文の中での働き（役割）** を表す記号であり、主語になるかたまりは形が何であれ **X**、名詞を修飾するかたまりは **Y**、動詞や文を修飾するかたまりは **Z** と呼びます。

この区別がわからないと、「動名詞は名詞なのに、なぜ X？」「関係詞節は形容詞じゃないのに、なぜ Y？」「副詞節は副詞じゃないのに、なぜ Z？」と混乱しやすくなります。だから本書では、早い段階で **品詞と役割は別の軸** であること、そして **X / Y / Z というフレームを採用する理由** を明示します。詳しくは [第4章](#ch-4) を参照してください。

### 何を確定する作業か

英文を読む・聞くときに、本書が求めるのは次の**2つだけ**です。

1. このかたまりは **X / Y / Z** のどれか
2. このかたまりは **どこにかかっている** か

**Walk-through の例**

`She read the book slowly.`

| かたまり | 役割 | かかっている先 |
|---------|------|--------------|
| She | X（主語） | 文の主役 |
| read | V（動詞） | — |
| the book | X（目的語） | read にかかる |
| slowly | Z（副詞） | read にかかる（ゆっくり読んだ） |

これが各位置で確定すれば、文の意味は再構成できます。「slowly が book にかかっている」のではなく **read にかかっている** — この一点を確定するだけで、意味の取り方が変わります。

### この章のまとめ

- 英文理解の焦点は **単語のラベル付け** ではなく **係り受けの確定**
- 文の理解は単語と論理の橋
- 各かたまりについて **X/Y/Z のどれか** と **何にかかるか** の2点を押さえる

## 第2章　英語と日本語の構造差

> アプリ該当：全Stepで意識する

### この章で扱うこと

英語学習で最も大きな壁は、たいてい「単語がわからない」ことだけではありません。**日本語と英語で、文の組み立て方そのものが違う**ことも、大きな壁になります。

この章では、その構造の違いを整理します。ここを押さえると、後の章やアプリの練習で「なぜ英語はこの語順なのか」が見えやすくなります。

### 一言での違い

まず、全体像だけ先に押さえておきましょう。

> 英語は **核（いちばん大事な言葉）を先に出して、説明を後ろに足す**（head-initial）  
> 日本語は **説明を前に積んで、核を最後に出す**（head-final）  
> つまり、**情報の並べる方向が逆**です。

たとえば「本」について話すとき：

- **日本語**：「私が昨日買った**本**」→ 説明が先、**本**という核が最後
- **英語**：「**the book** that I bought yesterday」→ **the book** という核が先、説明が後

同じ内容でも、**どこから読み始めて、どこで意味が確定するか**が違います。

### 4つの具体的な差

ここからは、その「方向の違い」が実際の英文でどう現れるかを、4つのポイントに分けて見ていきます。

---

**(1) 後置修飾 — 名詞の説明が「前」か「後」か**

日本語で名詞を修飾（説明）するとき、修飾語は**名詞の前**に置きます。英語では、多くの修飾が**名詞の後**に来ます。これを **後置修飾** と呼びます。

| 言語 | 例 | 構造 |
|------|----|----|
| 英語 | the book [that I bought yesterday] | 名詞が先、説明が後 |
| 日本語 | [私が昨日買った] 本 | 説明が先、名詞が後 |

**読み方のイメージ**

- **日本語**：「私が昨日買った … 本」と、**最後の「本」まで読んで初めて「何の話か」が確定**する読み方になりがちです。
- **英語**：「the book …」と、**最初に「本の話だ」と分かる**。そのあと `[that I bought yesterday]`（私が昨日買った）という説明が後ろから付いてきます。

つまり、日本語脳には「最後の名詞まで待つ」癖がつきやすい一方、英語は **「名詞 → 後ろに説明が続く」** という流れを前提に読まないと、途中で意味が崩れてしまいます。

**もう1つ例を見る**

| 日本語 | 英語 |
|--------|------|
| ピアノを弾いている**少女** | a girl **playing the piano** |
| 机の上の**本** | the book **on the desk** |

日本語は `[説明] + 名詞`、英語は `名詞 + [説明]` です。翻訳するときも、英語では **名詞を先に置いて、説明を後ろに回す** 必要があります。

---

**(2) 動詞の位置 — 「誰が・何をした」がいつ分かるか**

英語と日本語では、**動詞（何をしたか）が文のどこに来るか**も大きく違います。

**英語の場合**

文のはじめのほうで、主語と動詞が並びます。

```
I bought ...
↑   ↑
誰が  何をした
```

`I bought` だけでも、すでに **「私が買った」** という骨格（だれが・どうした）が確定します。そのあとに `a book yesterday` などの詳細が続き、**左から順に意味が固まっていきます**。

**日本語の場合**

同じ内容でも、動詞は文の**最後**に来ることが多いです。

```
私が  昨日  本を  買った
↑              ↑
誰が           何をした（ここで初めて確定）
```

最後まで読まないと、「結局何をしたのか（買ったのか、読んだのか、見たのか）」が分かりません。

**読解戦略**

> 英語は **最後まで待たない**。左から順に意味を固めていく。

英文を読むときは、「文末の動詞が来るまで保留する」のではなく、**最初に出てきた主語と動詞で骨格を先に固定**しましょう。そのうえで、後ろに続く修飾や詳細を足していく読み方が、英語には合っています。

---

**(3) 関係詞という明示マーカー — 「ここから説明が始まる」合図**

**(1) の後置修飾** では、英語は名詞の**後ろ**に説明を置きます。その「説明が始まった」ことを示す**合図**が、関係詞です。

**日本語の場合**

修飾語を名詞の前に置きますが、**「ここから説明です」という印は付きません**。

- 「**昨日買った**本」→ 「昨日買った」が説明だと、文脈から判断する

**英語の場合**

名詞の後ろに説明を置くとき、**`who` / `which` / `that`** などの関係詞で **「ここから後ろは、その名詞の説明です」** と明示します。

- `the book **that** I bought yesterday`
- `a man **who** speaks English`
- `the city **where** I live`

`that` や `who` を見たら、「この名詞について、後ろに説明が続く」と読み取れます。

**なぜ大事か**

日本語では「名詞の前＝説明」と決まっているので、印がなくても読めます。英語では説明が**後ろ**に来るうえ、節（小さな文）になることも多いので、**この合図を素早く拾えるかどうか**が、読むスピードや理解の正確さを左右します。関係詞に慣れる練習は、英語読解の近道になります。

---

**(4) 省略しない情報 — 英語は「文脈で分かるだろう」を表に出さない**

日本語は、文脈や状況から分かる情報を**省略しても**自然な文になります。英語は、日本語なら省略しがちな情報も**文の中に必ず表す**ことが多いです。

| 情報 | 英語 | 日本語 |
|------|------|--------|
| 主語 | 必須 | 省略可（「行きます」だけでOK） |
| 数（複数）| dogs（-s 必須）| 犬（単数・複数で形が変わらない）|
| 冠詞 | a / the を選ぶ | 冠詞に相当するものがない |
| 前置詞 | 名詞の前に置く（in the room）| 助詞が名詞の後ろに付く（部屋**に**）|

**具体例**

| 日本語 | 英語で必ず出すもの |
|--------|-------------------|
| （私は）行きます | **I** go |
| 犬がいる | **A** dog / **The** dog（冠詞） |
| 犬が3匹いる | **Three dogs**（複数形 -s） |
| 部屋にいる | **in** the room（前置詞） |

日本語では「誰が」「何匹か」「どの犬か」を、前後の文脈に任せて省略できます。英語は、**日本語が文脈に委ねる情報を、できるだけ文の表面に出す**言語です。そのぶん単語数は増えますが、1文だけ読んでも意味が取りやすくなる、という特徴があります。

---

### この章のまとめ

4つの差を並べると、こうなります。

1. **修飾の位置**：日本語は名詞の前、英語は名詞の後（後置修飾）
2. **動詞の位置**：英語は左から骨格が確定、日本語は文末で確定
3. **関係詞**：英語だけが「説明の始まり」を `who / which / that` で合図する
4. **省略**：英語は主語・数・冠詞・前置詞などを省略しにくい

英語を日本語と同じ「最後まで待つ」読み方で処理すると崩れやすいので、**核を先に拾い、左から意味を積み上げる**——この感覚を、以降の章でも意識してください。

## 第4章　品詞 — 8つの主役と機能語

> アプリ該当：Step 1（前提）

### この章で扱うこと

英文を分解するときの**最初の部品**が品詞です。8種類すべてを知ったうえで、本書ではそのうち **3つを主役（X / Y / Z）** として扱います。

### 8つの品詞

英語の品詞は、大きく8つに分類されます。それぞれ「文の中でどんな働きをするか」が決まっています。

| 品詞 | 英名 | 働き | 例 |
|------|------|------|----|
| 名詞 | Noun | もの・こと・人を表す | dog, water, love |
| 代名詞 | Pronoun | 名詞の代わり | I, you, it, this |
| 動詞 | Verb | 動作・状態を表す | run, is, have |
| 形容詞 | Adjective | 名詞を説明する | big, happy, red |
| 副詞 | Adverb | 動詞・形容詞・文を説明する | slowly, very, here |
| 前置詞 | Preposition | 名詞の前で関係を作る | in, on, at, with |
| 接続詞 | Conjunction | 語や文をつなぐ | and, but, because |
| 冠詞 | Article | 名詞の前の標識 | a, the |

**覚え方のヒント**

- **名詞・動詞・形容詞・副詞** → 意味の「主役」になりやすい
- **前置詞・接続詞・冠詞** → つなぎ・標識の役割（単独では文の主役にならない）

### 主役は3つだけ

8つのうち、意味を直接担う **主役は3つ** です。本書ではこれを **X / Y / Z** と呼びます。

| 記号 | 役割 | 元の品詞（代表例） |
|------|------|------------------|
| **X** | 名詞役 | 名詞・代名詞（ほか動名詞句・that節なども X になりうる） |
| **Y** | 形容詞役 | 形容詞（ほか前置詞句・関係詞節なども Y になりうる） |
| **Z** | 副詞役 | 副詞（ほか副詞節・前置詞句なども Z になりうる） |

---

### なぜ品詞名ではなく X / Y / Z を軸にするのか

ここが本書の**設計思想の核心**です。品詞を学んだあとに X / Y / Z が出てくると「また別の分類？」と感じるかもしれません。しかし、品詞と X/Y/Z は**別の軸**であり、英文を**構造として読む**ためには、後者の軸が必要です。

#### まず知っておいてほしいこと — X / Y / Z は本書の造語です

**X・Y・Z は、一般的な文法用語ではありません。** 教科書や辞書に載っている固定の呼び名でもなく、本ガイドと英文構造トレーナーで使う**独自の記号**です。

| 本書の記号 | 日本語での呼び方 | 近いが同一ではない概念 |
|-----------|----------------|---------------------|
| **X** | 名詞役 | Noun（名詞）・名詞相当語 |
| **Y** | 形容詞役 | Adjective（形容詞）・連体修飾 |
| **Z** | 副詞役 | Adverb（副詞）・副詞相当語 |

なぜわざわざ造語を使うのか——理由は3つあります。

1. **品詞名（Noun / Adjective / Adverb）だけでは、句や節の働きを表現しきれない** から
2. **「名詞役」「形容詞役」と毎回書くより、X / Y / Z の方が記号として扱いやすい** から（アプリの構造分解表示とも対応）
3. **品詞の用語と混同しない** ため——「名詞だから X」と短絡せず、「文の中で名詞と同じ仕事をしているから X」と考える必要がある、と読者に伝えたい

本書では、記号（X/Y/Z）と日本語名（名詞役・形容詞役・副詞役）を**セットで**使います。記号はラベル、日本語名は意味の説明です。

#### なぜ3つに簡略化するのか

英文の構造を読むとき、毎語ごとに8品詞を当てはめるだけでは、長い文で処理が追いつきません。本書は、文の意味を組み立てるうえで本当に必要な問いを、**3つに絞り込んでいます**。

| 簡略化 | 内容 |
|--------|------|
| 8品詞 → 3役割 | 修飾・骨格の埋め方の観点で、主に X / Y / Z の3分類に集約 |
| 多様な形 → 同じ記号 | 単語・句・節のどれでも、同じ役割なら同じ記号（X など） |
| 複雑な文 → 同じ手順 | 「X/Y/Z のどれか」→「何にかかるか」を、ネストの中でも繰り返す |

たとえば `People who work with me work hard when an issue arises.` では、品詞だけを追うと10語以上のラベル付けが必要です。X/Y/Z で見ると、

- 骨格：`People work hard`
- Y：`who work with me`（People を修飾）
- Z：`when an issue arises`（work hard を修飾）

と、**3層の役割**に整理できます。これが「簡略化してわかりやすくする」ということの意味です。

#### 品詞と役割は、問いが違う

品詞は **1語の種類** をラベル付けします。  
X / Y / Z は **かたまりが文の中で担う仕事** をラベル付けします。

英文理解で必要なのは後者です。X・Y・Z それぞれについて、同じ構図で見比べます。

**X（名詞役）の例 — `Playing tennis` が文頭にあるとき**

| 軸 | 問い | 答え |
|----|------|------|
| **品詞** | この語は何詞か？ | playing＝動詞の-ing形、tennis＝名詞 |
| **X** | このかたまりは文の中で何をしているか？ | **Playing tennis** 全体が **X（主語）** |

**Y（形容詞役）の例 — `the book on the desk` の `on the desk`**

| 軸 | 問い | 答え |
|----|------|------|
| **品詞** | この語は何詞か？ | on＝前置詞、the desk＝名詞句 |
| **Y** | このかたまりは文の中で何をしているか？ | **on the desk** 全体が **Y**（**book** を修飾） |

**Z（副詞役）の例 — `She studied in the library` の `in the library`**

| 軸 | 問い | 答え |
|----|------|------|
| **品詞** | この語は何詞か？ | in＝前置詞、the library＝名詞句 |
| **Z** | このかたまりは文の中で何をしているか？ | **in the library** 全体が **Z**（**studied** を修飾） |

骨格を読むとき、「ここは主語の枠（X）」「ここは名詞の説明（Y）」「ここは動詞の説明（Z）」と先に決めれば、中身が単語1つか句か節かは**二の次**に処理できます。

#### なぜこの説明が必要か — よくある混乱

品詞だけで英文を読もうとすると、X・Y・Z それぞれで壁にぶつかります。

| 混乱の対象 | 品詞だけで見たとき | 役割（X/Y/Z）で見たとき |
|-----------|------------------|----------------------|
| **X** 動名詞 | 「-ing は動詞の活用形では？」 | 主語の位置 → **X** |
| **X** 代名詞 | 「名詞とは別の品詞」 | 主語・目的語の枠を埋める → **X** |
| **X** that節 | 「節の中に動詞がある。名詞ではない」 | 目的語の枠を埋める → **X** |
| **Y** 前置詞句 | 「on は前置詞。形容詞ではない」 | 名詞の直後で名詞を修飾 → **Y** |
| **Y** 関係詞節 | 「節だから形容詞ではない」 | 名詞の直後で名詞を説明 → **Y** |
| **Y** 不定詞句 | 「to read は準動詞」 | `a book to read` で book を修飾 → **Y** |
| **Z** 副詞節 | 「節だから副詞ではない」 | 動詞を修飾（いつ・なぜ・どう）→ **Z** |
| **Z** 前置詞句 | 「in は前置詞。副詞ではない」 | 動詞の周辺で様子・場所を説明 → **Z** |
| **Z** 不定詞句 | 「to see は準動詞」 | `went there to see her` で目的を説明 → **Z** |

本書が品詞名の代わりに X / Y / Z を使うのは、**「名詞・形容詞・副詞という品詞の単語だけ」を指しているのではない** からです。

---

#### X（名詞役）— なぜ「Noun（名詞）」ではないか

**X** は、**主語・目的語・補語の枠を埋めるかたまり** を指す造語です。

| かたまり | 品詞・形 | X としての働き |
|---------|---------|--------------|
| She | 代名詞 | 主語（X） |
| a book | 名詞句 | 目的語（X） |
| Playing tennis | 動名詞句 | 主語（X） |
| that he is honest | that節 | 目的語（X） |

「Noun」と言うと **名詞という品詞の1語** を連想しがちです。X と言えば、**骨格の名詞の枠を埋めているかたまり全体** を指す、と意図が伝わります。代名詞・動名詞句・that節を別々のルールで覚えるより、**「ここは X の枠」** と先に固定する方が、長い文で骨格を読みやすくなります。

---

#### Y（形容詞役）— なぜ「Adjective（形容詞）」ではないか

**Y** は、**名詞を修飾・説明するかたまり** を指す造語です。

| かたまり | 品詞・形 | Y としての働き |
|---------|---------|--------------|
| red | 形容詞 | `a red book` で book を修飾 |
| on the desk | 前置詞句 | `the book on the desk` で book を修飾 |
| that I bought | 関係詞節 | `the book that I bought` で book を修飾 |
| to read | 不定詞句 | `a book to read` で book を修飾 |

英語では、名詞の修飾は **単語の形容詞だけ** に限りません（第2章の後置修飾）。`on the desk` は品詞上は「前置詞＋名詞」ですが、文の中では **book を説明している** — つまり **形容詞と同じ仕事** をしています。

「Adjective」と言うと **red のような1語** だけを指す印象が強く、`that I bought` のような節を形容詞の延長として読む習慣がつきにくい。**Y（形容詞役）** という造語は、「形容詞と**同じ仕事**をするかたまり」を明示するためのラベルです。

---

#### Z（副詞役）— なぜ「Adverb（副詞）」ではないか

**Z** は、**動詞・形容詞・文全体を修飾するかたまり** を指す造語です。

| かたまり | 品詞・形 | Z としての働き |
|---------|---------|--------------|
| slowly | 副詞 | `walked slowly` で walked を修飾 |
| in the library | 前置詞句 | `studied in the library` で studied を修飾 |
| when it rained | 副詞節 | `left when it rained` で left を修飾 |
| to see her | 不定詞句 | `went there to see her` で目的を説明 |

`when it rained` は節なので、品詞表では「副詞」に分類されません。しかし文の中では **left（いつ去ったか）** を説明しており、**副詞と同じ仕事** をしています。

「Adverb」と言うと **slowly のような1語** を連想しがちで、副詞節や前置詞句の修飾を体系化しにくい。**Z（副詞役）** は、「副詞と**同じ仕事**をするかたまり」をまとめて扱うための記号です。

---

#### このフレーム（X / Y / Z）を扱う意義

X / Y / Z を導入すると、英文を読む手順が一定します。

```
1. 骨格（動詞 V）を先に固定する
2. 残りのかたまりを X / Y / Z のどれかに分類する
3. それぞれが何にかかっているかを確定する
4. 節の中があれば、1〜3 を繰り返す（ネスト）
```

| フレームの利点 | 説明 |
|--------------|------|
| 問いが単純 | 各かたまりについて「X・Y・Z のどれか」だけ先に決めればよい |
| 品詞の壁を越える | 動名詞・関係詞節・前置詞句を、別ルールごとに暗記しなくてよい |
| アプリと一致 | 英文構造トレーナーの色分け（青・橙・紫）と同じ考え方 |
| 長文に強い | ネストが深くても、同じ X/Y/Z の問いを繰り返せる |

造語に見えて抵抗があるかもしれませんが、**一度この3記号に慣れると、品詞ごとにバラバラだったルールが1本の線でつながる** 体験が得られます。それが本書が X / Y / Z という形で簡略化する理由です。

#### かたまりのサイズ（単語・句・節）とも組み合わせられる

本書は [第5章](#ch-5) 以降、かたまりを **単語・句・節** の3サイズで扱います。X / Y / Z は、この3サイズの**どれにも** 適用できます。

```
X = 名詞（単語）/ 動名詞句（句）/ that節（節）
Y = 形容詞（単語）/ 前置詞句（句）/ 関係詞節（節）
Z = 副詞（単語）/ 前置詞句（句）/ 副詞節（節）
```

品詞名だけでは「節が主語になる」「前置詞句が名詞を修飾する」といった構造を、一貫した記号で表せません。X / Y / Z は **サイズを問わず、文での働きで分類する** ための共通言語です。

#### 「役」と付ける意味

**役** とは、文の中で **どの枠を埋めているか・何を修飾しているか** という**機能** のことです。

- **X（名詞役）** ＝ 名詞が担うのと同じ仕事（主語・目的語・補語）をするかたまり
- **Y（形容詞役）** ＝ 形容詞が担うのと同じ仕事（名詞の修飾）をするかたまり
- **Z（副詞役）** ＝ 副詞が担うのと同じ仕事（動詞・文の修飾）をするかたまり

記号 X/Y/Z は短く書けるラベル、日本語の「名詞役・形容詞役・副詞役」はその意味の説明——この2つをセットで使うことで、造語でありながら直感的に理解できるようにしています。

#### 品詞の知識は捨てない

X / Y / Z を軸にするからといって、品詞が不要になるわけではありません。

| 学習の段階 | 役割 |
|-----------|------|
| 品詞（8つ） | 単語の種類を知る。**入門の地図** |
| X / Y / Z（3役割） | かたまりの文での働きを知る。**構造読解の地図**（本書の造語） |
| 機能語 | かたまりをつなぐ。**補助の部品** |

品詞は「部品の名前」、X/Y/Z は「部品の組み立て方」を表す記号です。本書は後者を主軸に解説を展開しますが、品詞表（上の8品詞）は**前提知識として残しています**。

#### まとめ — 本書が X / Y / Z をベースにする理由

1. **X / Y / Z は本書の造語** であり、品詞の別名ではない——文の中での**役割**を表す
2. 英文理解の目的は **係り受けの確定** であり、品詞ラベルだけでは足りない
3. **X** は Noun ではなく、主語・目的語・補語の枠を埋めるかたまり全体を指す
4. **Y** は Adjective ではなく、名詞を修飾するかたまり全体（句・節を含む）を指す
5. **Z** は Adverb ではなく、動詞・文を修飾するかたまり全体（句・節を含む）を指す
6. 3つに簡略化することで、長い文・ネストした文でも **同じ問い（X/Y/Z？何にかかる？）** を繰り返せる

この6点が、本書が品詞名ではなく X / Y / Z をベースにガイドを展開する理由です。

---

### なぜ主役は3つに絞るのか

英文の骨格は「だれが・何をした・何を（どうなった）」で構成されます。名詞役（X）が主語・目的語・補語を埋め、形容詞役（Y）が名詞を修飾し、副詞役（Z）が動詞や文全体を修飾します——この3役割で、文の大部分をカバーできます。

残り（前置詞・接続詞・冠詞・関係詞など）は **機能語（つなぎ・標識）** です。単独では主役になりませんが、かたまりをつなげたり、節の始まりを示したりする**不可欠な部品**です。

### 例文での確認

次の文を、品詞ごとに分解してみましょう。

> `I usually buy bread and butter at a local supermarket.`

| 語 | 品詞 | 本書での扱い |
|----|------|------------|
| I | 代名詞 | X（主語） |
| usually | 副詞 | Z（頻度） |
| buy | 動詞 | V（動詞） |
| bread・butter | 名詞 | X（目的語） |
| and | 接続詞 | 機能語 |
| at | 前置詞 | 機能語（句の始まり） |
| a | 冠詞 | 機能語 |
| local | 形容詞 | Y（名詞を修飾） |
| supermarket | 名詞 | X（前置詞句の中核） |

品詞を知ることは第一歩です。次の章以降では、**複数の語がつながった「かたまり」** の扱い方に進みます。

## 第5章　単語・句・節 — 大きさによる3分類

> アプリ該当：全Stepの前提

### この章で扱うこと

英文のかたまりは、**大きさ**によって3種類に分けられます。この分類が、後の X / Y / Z の判定やネストの理解の土台になります。

### 判定軸は1つ

分類の判定軸は、次の**1つだけ**です。

> 中に「S＋V」（主語＋動詞）のセットがあるか？

| 種類 | 定義 | 例 |
|------|------|----|
| 単語（word）| 1語 | morning |
| 句（phrase）| 2語以上・S+V **なし** | in the morning |
| 節（clause）| 2語以上・S+V **あり** | when I was young |

**S+V とは**

「主語＋動詞」がセットで入っていれば、それは**小さな文**と同じ構造を持つ節です。入っていなければ、句（または単語）です。

### 例

- `in the morning` → 中に "I am" のような S+V がない → **句**
- `when I was young` → 中に "I was" という S+V がある → **節**

**もう1つ比較**

| かたまり | S+V がある？ | 分類 |
|---------|-------------|------|
| on the desk | なし | 句 |
| that I bought | あり（I bought） | 節 |
| slowly | 1語 | 単語 |

### なぜこの軸が重要か

句と節では、**文の中での扱い**が変わります。

- **句**は、名詞の後ろや動詞の周りに付く「修飾のかたまり」として機能します
- **節**は、内部が小さな文として成立しているため、**節の中にもまた X / Y / Z の構造が再帰的に現れます**（→ 第15章）

たとえば `when I was young` という節の中には、`I`（X）と `was young`（V + C）という骨格があります。長い英文は、この「節の中にまた節」という入れ子（ネスト）が積み重なっていきます。

## 第6章　動詞の5変化

> アプリ該当：Step 3 の前提

### この章で扱うこと

英語の動詞は、文脈に応じて**5つの形**に変化します。この5変化は、時制・相・態・準動詞（不定詞・動名詞・分詞）**すべての材料**になります。ここが脆弱だと、第9章以降が崩れます。

### 動詞が取る5つの形

`play` を例に、5つの形を見てみましょう。

| 形 | 名称 | 例（play） |
|----|------|-----------|
| 原形 | base | play |
| 三単現 | 3rd-person singular | plays |
| 過去形 | past | played |
| 過去分詞 | past participle | played |
| 現在分詞 | -ing form | playing |

**三単現とは**

主語が **he / she / it** など3人称単数のとき、現在形の動詞に **-s** を付けます。

- I play / He **plays**
- They go / She **goes**

### これらが組み合わさって何を作るか

5つの形は、それぞれ決まった場面で使われます。

| 形 | 使われる場面 |
|----|------------|
| 原形 | 助動詞の後・命令文・不定詞（to play） |
| 三単現 | 主語が3人称単数の現在形 |
| 過去形 | 過去のことを述べる |
| 過去分詞 | 完了形（have + 過去分詞）・受動態（be + 過去分詞）・分詞の形容詞用法 |
| 現在分詞 | 進行形（be + -ing）・動名詞・分詞の形容詞用法・分詞構文 |

**組み合わせの例（play）**

| 文 | 使われている形 |
|----|--------------|
| He **plays** tennis. | 三単現 |
| He **played** tennis. | 過去形 |
| He **has played** tennis. | 過去分詞（完了） |
| He **is playing** tennis. | 現在分詞（進行） |
| He wants **to play** tennis. | 原形（不定詞） |

5変化は **時制・相・態・準動詞すべての材料** となります。Step 3 では、この形の使い分けを集中的に練習します。

### 不規則動詞

`play` のように `-ed` で過去形・過去分詞が作れる動詞（規則動詞）以外に、**形が不規則に変わる動詞**があります。代表的なものを覚える必要があります。

```
be    – was/were – been
have  – had     – had
do    – did     – done
go    – went    – gone
see   – saw     – seen
take  – took    – taken
write – wrote   – written
read  – read    – read  （綴りは同じ、発音だけ変わる）
```

**覚え方**

- 原形・過去形・過去分詞の3つをセットで覚える
- `read` のように、綴りは同じでも発音が変わる動詞に注意

# 第3部　骨格を作る

## 第7章　5つの文型

> アプリ該当：Step 2（前提）

### この章で扱うこと

英文の**骨格**は、5つの基本パターン（文型）のいずれかに当てはまります。骨格を先に固定することが、長い文を読む第一歩です。

### 文型一覧

| 文型 | 構造 | 例 | 意味のイメージ |
|------|------|-----|--------------|
| 第1 | S + V | He runs. | 彼は走る |
| 第2 | S + V + C | He is happy. | 彼は幸せだ |
| 第3 | S + V + O | She bought a pen. | 彼女はペンを買った |
| 第4 | S + V + O + O | He gave her a book. | 彼は彼女に本をくれた |
| 第5 | S + V + O + C | They elected him president. | 彼らは彼を社長に選んだ |

**S / V / O / C とは**

- **S**（Subject）= 主語（だれが）
- **V**（Verb）= 動詞（何をした）
- **O**（Object）= 目的語（何を）
- **C**（Complement）= 補語（〜である・〜になる）

### O と C の見分け方

第3文型以降で迷いやすいのが、**O（目的語）と C（補語）の違い**です。

- **O（目的語）**：主語と **「別物」** を指す名詞
  - She bought **a pen**. → "she" と "a pen" は別物（彼女 ≠ ペン）
- **C（補語）**：主語または目的語を **「説明」** する語
  - He is **happy**. → "he" の状態を説明（彼＝幸せな状態）
  - They elected him **president**. → "him" の役割を説明（彼＝社長になった）

**見分けのコツ**

「=」でつなげられるかどうか。補語 C は「主語（または目的語）＝C」と言い換えられることが多いです。

- He is happy. → He = happy ✓
- She bought a pen. → She = a pen ✗

### 補語 C の中身

C には2種類あります（→ 第10・11章で詳述）：

- **形容詞（Y）**：He is happy.
- **名詞（X）**：He became a doctor.

→ 文型公式では `C = X または Y` と表記します。

## 第8章　動詞のタイプと文型の対応

> アプリ該当：Step 2 の補完

### この章で扱うこと

文型（S + V + O / C）は、**動詞の種類**によって決まります。英文を読むときは **まず動詞を見つける** — これが骨格解析の起点です。

### 動詞のタイプは文型を決める

文型を選ぶのは話者ではなく **動詞そのもの** です。各動詞は、取れる文型が決まっています。

| 動詞のタイプ | 取れる文型 | 代表例 |
|------------|-----------|--------|
| 完全自動詞 | SV | run / sleep / arrive |
| 不完全自動詞（連結動詞）| SVC | be / become / seem / look / feel |
| 完全他動詞 | SVO | eat / read / want |
| 授与動詞 | SVOO | give / tell / show / send |
| 不完全他動詞 | SVOC | make / find / call / elect |
| 知覚動詞 | SVO + 原形/-ing/-ed | see / hear / feel / watch |
| 使役動詞 | SVO + 原形/to do/-ed | make / have / let / get |

**タイプごとのイメージ**

| タイプ | 動詞を見たら予測できること |
|--------|------------------------|
| 完全自動詞 | 目的語は来ない（He runs. で完結） |
| 連結動詞 | 後ろに C（補語）が来る（He is happy.） |
| 完全他動詞 | 後ろに O（目的語）が来る（She read a book.） |
| 授与動詞 | O が2つ（He gave her a book.） |
| 不完全他動詞 | O + C（They elected him president.） |

### 動詞起点で読む

英文を読むときは **まず動詞を見つける**。動詞が決まると、その後ろに何が来るかが予測できます。

> 例：`become` を見たら → 後ろに C（補語）が来ると予測 → "a doctor" は X 役の C と判定

**Walk-through**

`He became a doctor.`

1. 動詞 `became` を見つける → 連結動詞（SVC）と判断
2. 左に主語 `He`（X）
3. 右に補語 `a doctor`（X 役の C）が来るはず → 予測どおり

動詞のタイプを知っていると、**まだ文を読み終わっていなくても**、後ろに何が来るかを予測できます。

## 第9章　動詞が背負う4情報

> アプリ該当：Step 3 本体

### この章で扱うこと

英語の動詞は、**4種類の情報**を同時に背負います。Step 3 では、日本語の「いつ・どんな様子で・する/される・どのくらい確実か」を、この4情報のどの形に対応させるかを練習します。

### 動詞は4つの情報を背負う

| 情報 | 軸数 | 内容 |
|------|------|------|
| 時制（Tense）| 3 | 現在 / 過去 / 未来（will） |
| 相（Aspect）| 4 | 単純 / 進行（be+-ing）/ 完了（have+pp）/ 完了進行（have been+-ing）|
| 態（Voice）| 2 | 能動 / 受動（be+pp）|
| 法（Modality）| n | 直説法 / 仮定法 / 助動詞（can/must/may/will/should...）|

**日本語との対応イメージ**

| 日本語の手がかり | 英語で表現する情報 |
|-----------------|-------------------|
| いつ（今・昨日・すでに） | 時制 |
| どんな様子で（〜している・〜していた） | 相（進行・完了） |
| する / される | 態（能動・受動） |
| できる・しなければ・〜だろう | 法（助動詞） |

### 時制 × 相 = 12形

時制（3）と相（4）を組み合わせると、**12形**になります（`play` を例に）。

| | 単純 | 進行 | 完了 | 完了進行 |
|--|------|------|------|---------|
| 現在 | plays | is playing | has played | has been playing |
| 過去 | played | was playing | had played | had been playing |
| 未来 | will play | will be playing | will have played | will have been playing |

**読み方の例**

- `is playing` → 現在 + 進行 → 「今、〜している」
- `has played` → 現在 + 完了 → 「すでに〜した（経験・完了）」
- `had played` → 過去 + 完了 → 「（その時点で）すでに〜していた」

### 態を加えると24形

上記12形それぞれに、**受動態（be + 過去分詞）** が加わります。さらに助動詞・否定が組み合わさると、形のバリエーションは増えていきます。

**例**

| 能動 | 受動 |
|------|------|
| He wrote the report. | The report was written by him. |
| She is reading a book. | A book is being read by her. |

### 何を訓練するか

Step 3 では、日本語の「いつ・どんな様子で・する/される・どのくらい確実か」を、上記マトリクスの**どの形に対応させるか**を練習します。

- 「昨日買った」→ 過去形
- 「今読んでいる」→ 現在進行形
- 「すでに書かれている」→ 現在完了受動
- 「3か国語を話せる」→ 助動詞 can + 原形

# 第4部　3つの主役 X / Y / Z

## 第10章　X — 名詞役

> アプリ該当：全Stepで使用、Step 5・6 で集中訓練

### この章で扱うこと

**X（名詞役）** は、文の骨格で **S（主語）・O（目的語）・C（補語の一部）** になれるかたまりです。単語1つから節まで、さまざまな形を取ります。

### X の働き

X は「名詞として機能するかたまり」です。次の位置に入れます。

- **S（主語）**：`She` bought a pen.
- **O（目的語）**：She bought `a pen`.
- **C（補語・名詞）**：He became `a doctor`.

### X が取る形

X は、大きさによって次の形を取ります。

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 名詞・代名詞 | book / I |
| 句 | 名詞句 | a business book |
| 句 | 動名詞句（-ing）| Playing tennis is fun. |
| 句 | 不定詞句（to do）| To play tennis is fun. |
| 句 | 疑問詞+不定詞 | I don't know what to do. |
| 節 | that節 | I know that he is honest. |
| 節 | whether/if節 | I wonder whether it is true. |
| 節 | wh節（疑問詞節）| I know what you said. |

**サイズの見分け方**

- 1語 → 単語
- 2語以上で S+V なし → 句
- 2語以上で S+V あり → 節（第5章）

### 節の3パターンの理由

名詞節（X として機能する節）は、**もともと1文として成立していた文** を、別の文の中に「名詞のかわり」として埋め込んだものです。

表の「元になる文」とは、**節の中身を単独の文に戻したときの形** のことです。つまり、

```
I know [that he is honest].
              └─ 中身だけ取り出すと → He is honest.（平叙文）
```

という関係です。節の種類は、**中身がもともとどんな文だったか** で決まります。

| 節 | 元になる文 | 埋め込むと |
|----|-----------|-----------|
| that節 | 平叙文 → He is honest. | I know **that he is honest**. |
| whether/if節 | Yes/No疑問文 → Is it true? | I wonder **whether it is true**. |
| wh節 | 情報疑問文 → What did you say? | I know **what you said**. |

3つで名詞節を **MECE（漏れなくダブりなく）に網羅** します。英語の文を名詞節にするとき、元の形は必ずこの3種類のどれかになります。

---

#### 平叙文と that節 — どうつながるか

ここがいちばんつまずきやすいポイントです。

**平叙文** とは、「〜です／〜だ」という**普通の肯定文**のことです。

```
He is honest.   （彼は正直だ）
```

この文を、別の文の **目的語（O）の位置** に埋め込みたいとします。日本語では：

- 彼は正直だ → **私は知っている**（「彼が正直だ」という**内容全体**が「知る」の対象）

英語でも同じで、`know` の目的語になるのは単語1つではなく、**「彼が正直だ」という内容のかたまり** です。

**埋め込みの手順**

| 段階 | 英文 | 説明 |
|------|------|------|
| ① 元の平叙文 | He is honest. | 単独で意味が通る文 |
| ② 別の文に組み込む | I know ___ . | know の後ろ（目的語の枠）に入れる |
| ③ 接続語 that を付ける | I know **that** he is honest. | 「ここから先が内容です」と合図 |
| ④ 人称を調整 | … that **he** is honest | 文の主語が I なので he に（必要なら） |

```
I know [that he is honest].
  │      └──── X（名詞節・目的語）────┘
  │            中身 ＝ He is honest. と同じ内容
  └─ V
```

**that の役割**

- `that` は **「ここから後ろが、前の動詞の内容（名詞節）です」** という合図
- 平叙文 `He is honest.` の骨格（he + is + honest）は**そのまま残る**
- 文頭に `that` が付くだけで、**1文が名詞のかわりに使える** ようになる

**もう1つ例**

| 平叙文（単独） | that節として埋め込んだ文 |
|--------------|------------------------|
| She will come. | I think **that she will come**. |
| He was tired. | She said **that he was tired**. |

どちらも、**中身の骨格は平叙文のまま**、`that` で「名詞節です」と示しているだけです。

> **覚え方**：平叙文をそのまま中に入れて、先頭に `that` を付けたもの ＝ **that節（X）**

---

#### Yes/No疑問文と whether/if節

**Yes/No疑問文** は、「〜ですか？」と**はい／いいえ**で答える疑問文です。

```
Is it true?   （それは本当ですか？）
```

これを名詞節にすると、「それが本当かどうか」という**内容全体**が名詞のかわりになります。

| 段階 | 英文 |
|------|------|
| ① 元の疑問文 | Is it true? |
| ② 語順を平叙文に戻す | it is true（疑問文 → 平叙文の語順） |
| ③ whether / if を付ける | whether it is true |
| ④ 別の文に埋め込む | I wonder **whether it is true**. |

```
I wonder [whether it is true].
           └──── X（名詞節）────┘
                 中身 ＝ Is it true? と同じ問い
```

`whether` / `if` は that と同様、**「ここから名詞節が始まります」** という合図です。中身は Yes/No 疑問文を平叙文の語順に直した形になります。

---

#### 情報疑問文と wh節

**情報疑問文** は、what / who / where などで**具体的な情報**を聞く疑問文です。

```
What did you say?   （あなたは何と言いましたか？）
```

これを名詞節にすると、「何を言ったか」という**内容**が名詞のかわりになります。

| 段階 | 英文 |
|------|------|
| ① 元の疑問文 | What did you say? |
| ② 語順を平叙文に戻す | you said what → what you said |
| ③ 別の文に埋め込む | I know **what you said**. |

```
I know [what you said].
         └─ X（名詞節）─┘
            中身 ＝ What did you say? と同じ問い
```

疑問詞（what）が **節の先頭に残る** のが、that節との違いです。

- **that節**：平叙文 ＋ `that`（疑問の形はしない）
- **wh節**：情報疑問文を平叙文の語順に直し、疑問詞を先頭に残す

---

#### 3パターンの見分け方 — まとめ

名詞節を見たとき、**中身だけ取り出して** 次の問いを立てます。

| 問い | 節の種類 | 例（中身） |
|------|---------|-----------|
| 単なる平叙文に戻せる？ | **that節** | He is honest. |
| Yes/No 疑問文に戻せる？ | **whether/if節** | Is it true? |
| 情報疑問文に戻せる？ | **wh節** | What did you say? |

**Walk-through：`I know that he is honest.`**

1. `that he is honest` が名詞節（X）だと分かる
2. `that` を取ると `he is honest` — 平叙文として自然 ✓
3. → **that節**（平叙文が元）

**Walk-through：`I wonder whether it is true.`**

1. `whether it is true` が名詞節（X）
2. 語順を疑問形に戻すと `Is it true?` — Yes/No 疑問文 ✓
3. → **whether節**（Yes/No 疑問文が元）

この「中身を単独の文に戻す」作業が、平叙文と that節の関係をつかむ近道です。

## 第11章　Y — 形容詞役と修飾の方向

> アプリ該当：Step 4・5 で集中訓練

### この章で扱うこと

**Y（形容詞役）** は、**名詞を修飾する**（飾る・説明する）かたまりです。第2章で学んだ「後置修飾」の中心が、ここに集まります。

### Y の働き

Y は「形容詞として機能するかたまり」です。常に **名詞にかかります**。

- `an **interesting** book` → interesting が book を修飾
- `the book **that I bought**` → 関係詞節全体が book を修飾

### Y が取る形

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 形容詞 | an **interesting** book |
| 単語 | 分詞単独 | a **broken** window / a **sleeping** baby |
| 句 | 前置詞句 | the book **on the desk** |
| 句 | 不定詞句 | a book **to read** |
| 句 | 現在分詞句 | a girl **playing the piano** |
| 句 | 過去分詞句 | a book **written by her** |
| 句 | 同格 | my friend **Tom** |
| 句 | 形容詞句 | a **very tall** man |
| 節 | 関係代名詞節 | the book **that I bought** |
| 節 | 関係副詞節 | the city **where I live** |
| 節 | 同格節 that | the fact **that he lied** |

### 修飾の方向

英語には、**修飾語を名詞の前に置くか後ろに置くか** という重要なルールがあります。

| 修飾するもの | 配置 | 例 |
|------------|------|-----|
| 単語（形容詞・分詞単独）| **前置** | a **red** book |
| 句（前置詞句・分詞句・不定詞句など）| **後置** | the book **on the desk** |
| 節（関係詞節・同格節）| **後置** | the book **that I bought** |

**なぜこのルールがあるか**

- **1語**なら名詞の前に置ける（a red book）
- **2語以上のかたまり**（句・節）は名詞の**後ろ**に置く（the book on the desk）

これが第2章で学んだ「後置修飾」の具体ルールです。

**例外**：不定代名詞（something / anything / nobody）は、単語の形容詞でも**後置**します。

- ○ something **interesting**
- × interesting something

### 関係詞節と同格節の違い

`that` を使った後置修飾は、見た目がとても似ています。

```
the book that I read
the fact that he left
```

どちらも `名詞 + that + S + V` の形です。しかし **内部の構造がまったく違います**。混同すると訳し方も語順も崩れるので、ここを丁寧に分けます。

| 種類 | 空所（gap） | 役割 | 例 |
|------|-----------|------|-----|
| **関係詞節** | **あり** | 名詞の**中身を特定**する（どの本か） | the book **that I read** |
| **同格節 that** | **なし** | 名詞の**内容を言い換える**（どんな事実か） | the fact **that he left** |

どちらも Y（名詞を修飾）ですが、「that の後ろが名詞とどう関係するか」が違います。

---

#### 関係詞節 —「空所」を that が埋める（gap あり）

**関係詞節** は、節の中に **空所（gap）** があり、`that`（や which / who）が **先行詞（前の名詞）の代わり** にその空所を埋めます。

**例：`the book that I read`**

| 段階 | 文 | 説明 |
|------|-----|------|
| ① 2つの文に分ける | I read the book. / The book is interesting. | もともと別々に言える |
| ② 重なる語を空所に | I read ___. / The book is interesting. | `the book` が重複 |
| ③ that で空所を埋める | I read **that**. … **The book** that I read is interesting. | that ＝ the book |

```
the book [that I read ___].
         └─ Y（関係詞節）─┘
              that ＝ 空所（目的語の位置）
              「I read [???]」の ??? を the book で埋めている
```

**ポイント**

- `that` は **「the book の代わり」** として節の中に入る
- 節だけ取り出すと `I read ___` となり、**何かが足りない**（空所がある）
- 日本語の「**私が読んだ**本」＝ 名詞（本）の**特定**

**別の例**

| 関係詞節 | 分けた2文 | 空所の位置 |
|---------|----------|-----------|
| the man **who spoke** | The man spoke. / The man is my boss. | 主語（who spoke） |
| the city **where I live** | I live in the city. | 場所（where） |
| the book **(that) I bought** | I bought the book. | 目的語（that） |

> **覚え方**：関係詞節は「前の名詞を、後ろの節の中に差し込める」＝ **gap あり**

---

#### 同格節 that —「内容」そのものを述べる（gap なし）

**同格節 that** は、前の名詞が **「内容・事実・考え」** を表すとき、その**中身をそのまま言い換える**節です。節の中に空所はありません。

**例：`the fact that he left`**

| 段階 | 文 | 説明 |
|------|-----|------|
| ① 名詞の意味を確認 | the fact ＝ 「事実」という**抽象名詞** | 中身を説明する必要がある |
| ② 中身を平叙文で | He left. | これが「事実」の内容 |
| ③ that でつなぐ | the fact **that he left** | 「彼が去った」という事実 |

```
the fact [that he left].
         └─ Y（同格節）─┘
              he left は完成形（空所なし）
              「事実」の中身 ＝ 彼が去った、ということ
```

**ポイント**

- `he left` は **それだけで1文として完結** する（空所がない）
- `that` は **関係代名詞ではなく接続詞** の働き——「以下が fact の内容です」
- 前の名詞 `fact` を節の中に差し込もうとするとおかしい（× he left the fact）
- 日本語の「**彼が去った**という事実」＝ 名詞（事実）の**内容説明**

**同格節が付きやすい名詞**

fact / idea / news / belief / rumor / hope / thought / saying など、**「〜という内容」** を持つ名詞：

| 同格節 | 意味 |
|--------|------|
| the idea **that we should leave** | 私たちは去るべきだ、という考え |
| the news **that she won** | 彼女が勝った、という知らせ |
| the belief **that he is innocent** | 彼は無実だ、という信念 |

---

#### 見分け方 — 3つのチェック

形が似ているので、次の順番で確認します。

**チェック1：節の中に空所があるか？**

| 節だけ取り出す | 空所 | 判定 |
|--------------|------|------|
| I read ___ | あり（目的語がない） | 関係詞節 |
| he left | なし（完成形） | 同格節 |

**チェック2：2文に分けられるか？**

| 文 | 分け方 | 判定 |
|----|--------|------|
| the book that I read | I read the book. ✓ | 関係詞節 |
| the fact that he left | He left. ✓（ただし「事実」と「去った」は別の言い方） | 同格節 |

関係詞節は **前の名詞を後ろの節にそのまま戻せる**（I read **the book**）。  
同格節は **that の後ろが名詞の中身の言い換え** で、名詞を節に差し込むイメージではない。

**チェック3：前の名詞は「もの」か「内容」か？**

| 先行詞 | タイプ | 判定 |
|--------|--------|------|
| book / man / city | 具体的なもの・人・場所 | 関係詞節になりやすい |
| fact / idea / news | 抽象名詞（中身がある） | 同格節になりやすい |

---

#### Walk-through：2文を比較する

**関係詞節：`The book that I bought is interesting.`**

```
The book is interesting.     ← 骨格
    the book [that I bought]   ← Y。that I bought ＝ どの本か
         I bought the book.   ← 名詞を節に戻せる ✓
```

**同格節：`The fact that he lied surprised me.`**

```
The fact surprised me.         ← 骨格
    the fact [that he lied]    ← Y。that he lied ＝ どんな事実か
         He lied.              ← 事実の中身（完成形）✓
         × He lied the fact.   ← 名詞を節に戻せない ✗
```

---

#### なぜ混同しやすいか — 第10章の that節との関係

| that の種類 | 役割 | 例 |
|------------|------|-----|
| 名詞節の that（第10章） | 節全体が **X**（目的語など） | I know **that he is honest**. |
| 関係詞節の that | 節が **Y**（名詞を修飾）、gap あり | the book **that I read** |
| 同格節の that | 節が **Y**（名詞の内容）、gap なし | the fact **that he lied** |

3つとも `that` を使いますが、**文の中での働き** が違います（→ [第14章](#ch-14)）。見た目だけで判断せず、**空所の有無** と **前の名詞との関係** で見分けてください。

> **覚え方**
> - 関係詞節：「**どの** book？」→ 名詞を特定（gap あり）
> - 同格節：「**どんな** fact？」→ 名詞の中身を説明（gap なし）

## 第12章　Z — 副詞役と配置ルール

> アプリ該当：Step 4・6 で集中訓練

### この章で扱うこと

**Z（副詞役）** は、**動詞・形容詞・他の副詞・文全体** を修飾するかたまりです。Y が名詞を修飾するのに対し、Z は動詞や文を修飾します。

### Z の働き

Z は「副詞として機能するかたまり」です。かかる先は次のとおりです。

- **動詞**：walked **slowly**（ゆっくり歩いた）
- **形容詞**：**very** tall（とても背が高い）
- **他の副詞**：walked **very** slowly
- **文全体**：**Unfortunately**, she left.（残念ながら、彼女は去った）

### Z が取る形

| サイズ | 形 | 例 |
|--------|----|----|
| 単語 | 副詞 | walked **slowly** |
| 句 | 前置詞句 | studied **in the library** |
| 句 | 不定詞句 | went there **to see her** |
| 句 | 分詞構文 | **Walking home**, he met her. |
| 句 | 副詞句 | walked **very slowly** |
| 句 | 名詞句の副詞用法 | gets up at 6 **every morning** |
| 節 | 副詞節 | left **when it rained** |

### Z の3つの配置位置

副詞（Z）は、文の中で **3つの位置** に置けます。

```
① 前置（Front）：文頭・主語の前
② 中置（Mid）   ：主語直後・動詞直前（be / 助動詞の後）
③ 後置（End）   ：動詞句の後ろ（文末）
```

**配置の例**

| 位置 | 例 |
|------|-----|
| ① 前置 | **Yesterday**, she went home. |
| ② 中置 | She **always** goes home early. |
| ③ 後置 | She went home **yesterday**. |

### 種類別の好み

副詞の**種類**によって、置きやすい位置が決まっています。

| 種類 | 好みの位置 | 例 |
|------|-----------|-----|
| 様態 | ③ 後置（または②）| She walked home **slowly**. |
| 頻度 | ② 中置（強い）| She **always** goes home early. |
| 時 | ③ 後置 または ① 前置 | She went home **yesterday**. |
| 場所 | ③ 後置 | She waited **outside**. |
| 程度 | 修飾する語の直前 | She walked **very** slowly. |
| 文修飾 | ① 前置 または ② 中置 | **Unfortunately**, she left. |

### 重要なルール

**(1) 目的語を飛び越えない**

様態副詞（slowly など）は、**目的語の後ろ**に置きます。目的語を飛び越えて動詞の直後に置くと不自然です。

```
○ She read the book slowly.   ← 目的語の後ろ
× She read slowly the book.   ← 非自然
```

**(2) 複数の副詞が並ぶときの順**

複数の副詞が並ぶときは、次の順序が基本です。

> 様態 → 場所 → 時

```
She walked slowly (様態) home (場所) yesterday (時).
```

### 副詞節の細分

副詞節（Z として機能する節）は、**接続詞**によって意味が決まります。

| 種類 | 接続詞 |
|------|--------|
| 時 | when / while / before / after / until / since |
| 理由 | because / since / as / now that |
| 条件 | if / unless / in case |
| 譲歩 | although / though / even though |
| 目的 | so that / in order that |
| 結果 | so ... that / such ... that |
| 様態 | as / as if / as though |
| 比較 | than / as ... as |
| 場所 | where / wherever |

**例**

- `when it rained` → 時の副詞節
- `because he was tired` → 理由の副詞節
- `if it rains` → 条件の副詞節

## 第13章　3×3マトリクス — 役割 × 表現方法の全体像

> アプリ該当：本書の核

### この章で扱うこと

X / Y / Z の3役割と、単語・句・節の3サイズを組み合わせた **3×3マトリクス** が、本書の全体地図です。この表を頭に置くと、英文のどの部分も「どのセルに当てはまるか」で整理できます。

### マトリクス本体

|  | Word（単語）| Phrase（句・S+Vなし）| Clause（節・S+Vあり）|
|--|------------|--------------------|---------------------|
| **X** 名詞役 | 名詞・代名詞 | 名詞句・動名詞句・不定詞句・疑問詞+不定詞 | that節・whether/if節・wh節 |
| **Y** 形容詞役 | 形容詞・分詞単独 | 前置詞句・不定詞句・分詞句・同格・形容詞句 | 関係代名詞節・関係副詞節・同格節 that |
| **Z** 副詞役 | 副詞 | 前置詞句・不定詞句・分詞構文・副詞句・名詞句の副詞用法 | 副詞節（時/理由/条件/譲歩/目的/結果/様態/比較/場所）|

**読み方**

- 縦軸 = **役割**（X / Y / Z）
- 横軸 = **サイズ**（単語 / 句 / 節）
- たとえば「the book that I bought」は **Y × 節** のセル（関係代名詞節）

### 高タイパなパーツ

複数のセルに登場する形は **学習コスパが高い**（1つの形を覚えれば、複数の場面で使える）です。

| 形 | 出現するセル | 効果 |
|----|------------|------|
| 不定詞（to do）| X・Y・Z 全て | 1つの形で3役割をカバー |
| 前置詞句 | Y・Z 両方 | 1つの形で2役割 |
| -ing形 | X（動名詞）・Y（分詞）・Z（分詞構文）| 1つの形で3役割 |
| 分詞 | Y（後置修飾）・Z（分詞構文）| 1つの形で2役割 |

→ Step 4 で **準動詞（不定詞・動名詞・分詞）と前置詞句** を集中訓練するのは、ここに高タイパが集中しているからです。

**具体例：不定詞 to do**

| 役割 | 例 |
|------|-----|
| X | **To read** is fun. |
| Y | a book **to read** |
| Z | I went there **to read**. |

同じ `to read` でも、位置と働きで役割が変わります（→ 第14章）。

## 第14章　同じ形が複数の役割を持つ

> アプリ該当：全Stepで意識

### この章で扱うこと

第13章のマトリクスは「形」では **MECE（漏れなくダブりなく）** にはなりません。同じ形が、文の中の**位置と働き**によって、X / Y / Z の複数の役割を持つことが頻繁にあります。

### MECE にならない理由

たとえば `to read` という形は、主語にも、名詞の後置修飾にも、目的を表す副詞役にもなります。**形だけ見ても役割は決まりません**。

### 主要な「同形・複数役割」一覧

| 形 | X | Y | Z |
|----|---|---|---|
| 不定詞 to do | ○ | ○ | ○ |
| 前置詞句 | - | ○ | ○ |
| -ing | ○（動名詞）| ○（分詞）| ○（分詞構文）|
| that節 | ○（名詞節）| ○（同格節・関係詞節）| - |
| when / where 等 | ○（wh節）| ○（関係副詞節）| ○（副詞節）|

### 具体例で違いを見る

**不定詞 to do**

| 役割 | 例 | 判断のポイント |
|------|-----|--------------|
| X | **To read** is fun.（主語）| 文の主語の位置 |
| Y | a book **to read**（books を後置修飾）| 名詞の直後 |
| Z | I went there **to read**.（目的・副詞役）| 動詞の周辺・目的を表す |

**when 節**

| 役割 | 例 | 判断のポイント |
|------|-----|--------------|
| X | I know **when he left**.（名詞節・目的語）| 動詞の目的語の位置 |
| Y | the day **when he left**（day を後置修飾）| 名詞の直後 |
| Z | He left **when it rained**.（副詞節・時）| 動詞を修飾 |

### 結論

> **形ではなく、文中での位置と働きで役割が決まる**

これが英文解読の核心です。形を覚えるだけでは不十分で、各かたまりの **役割（X/Y/Z）** と **何にかかっているか** を判定する訓練が必要です。

## 第15章　ネストの再帰性

> アプリ該当：Step 5・6 で集中訓練

### この章で扱うこと

長い英文は、**かたまりの中にまたかたまりが入る**（ネスト）構造になっています。本書の中心原則の1つは、**X / Y / Z のルールが内部でも同じように適用される** ことです。

### 中心原則

> X / Y / Z のルールは句・節の **内部でも再帰的に適用される**

つまり、節の中身もまた一つの**小さな文**として X / Y / Z で解析できます。大きな文を分解したら、中の節に対して**同じ手順を繰り返す**イメージです。

### 例1：節の中のネスト

```
I know [that she slowly reads books at night].
         └──── 名詞節（X）────┘
              ├─ she（X・主語）
              ├─ slowly（Z・様態副詞・中置）
              ├─ reads books（V + X）
              └─ at night（Z・時の前置詞句・後置）
```

**読み方**

1. 外側の骨格：`I know ...` → know の目的語が `[that she slowly reads books at night]`
2. その節（X）の中を見る → 内部にも she（X）、reads（V）、books（X）、slowly（Z）、at night（Z）がある
3. 節の中を**別の文**として同じ手順で分解

### 例2：複数のかたまりのネスト

```
People [who work with me] work hard [when an issue arises].
        └──── Y（関係詞節）─┘         └──── Z（副詞節）────┘
```

**読み方**

1. 骨格：`People work hard`（人々は一生懸命働く）
2. `who work with me` = Y（関係詞節）→ **People** を後置修飾
3. `when an issue arises` = Z（副詞節）→ **work hard** を修飾
4. when節の内部にも `an issue`（X）+ `arises`（V）がネスト

これが本書のキーセンテンスの構造です。

### 句の中のネスト

節だけでなく、**句の中**にもネストがあります。

```
the book [on the desk]
          └─ 前置詞句（Y）
              └─ "the desk" = X（名詞）が句の中にネスト
```

`on the desk` という前置詞句（Y）の中に、`the desk`（X）が入っています。

### 読解上の意味

ネストは **「処理できるかたまりの長さと深さ」** が学習進度を決めます。

- Step 3：骨格 + 動詞情報（ネスト浅い）
- Step 4：句が付く（1段ネスト）
- Step 5：関係詞節（節のネスト）
- Step 6：Y + Z の複合ネスト（深い）

Step が進むほど、ネストの深さも増します。アプリの Step 6 は、このネストを正面から扱います。

## 第16章　冠詞・限定詞

> アプリ該当：Step 4 以降の名詞句構成

### この章で扱うこと

名詞（X）の前には、**冠詞・限定詞** などの機能語が付くことが多いです。これらは単独では文の主役にはなりませんが、名詞句を構成する**必須の部品**です。

### 名詞の前の標識

冠詞・限定詞は、名詞の前に置いて「どんな名詞か」を示します。

| 種類 | 例 |
|------|-----|
| 冠詞 | a / an / the |
| 指示詞 | this / that / these / those |
| 所有格 | my / your / his / her / its / our / their |
| 数量詞 | some / any / many / much / few / several / all |
| 数詞 | one / two / three ... |

**役割**

- これらは X / Y / Z の主役ではなく、**名詞句を形作る機能語**
- ただし名詞句を正確に読む・書くには欠かせない

### 名詞句の組み立て順

英語の名詞句は、だいたい次の順序で組み立てられます。

```
[限定詞] [数詞] [形容詞] 名詞 [後置修飾]
   the      two       tall      men     in the park
```

→ 一つの名詞句の中に **限定詞・形容詞（Y/Word）・名詞（X/Word の核）・修飾句（Y/Phrase）** がネストしています（→ 第15章・第21章）。

### a / the の使い分けの原理

| 冠詞 | 使う場面 |
|------|---------|
| **a / an** | 話し手と聞き手の共通認識にない、初出または不特定の名詞 |
| **the** | 話し手と聞き手の共通認識にある特定の名詞 |

**例**

- I bought **a** book. → どの本か特定していない（初出）
- I read **the** book you gave me. → 特定の本（共通認識）

詳細は別途学習が必要ですが、X 役の名詞句を作る上で必須の知識です。

## 第17章　前置詞

> アプリ該当：Step 4 で集中訓練

### この章で扱うこと

**前置詞**は、名詞と組み合わさって **Y または Z** に変換する「装置」です。同じ `in the park` でも、名詞の後ろなら Y、動詞の周辺なら Z になります。

### 前置詞の役割

前置詞は **名詞を Y または Z に変換する** 装置です。

```
in + the park（名詞・X）→ "in the park" = Y または Z
```

- 名詞 `the park` 単体 → X
- 前置詞 `in` が付く → 前置詞句 `in the park` → Y または Z（位置で決まる）

### 主な前置詞の分類

| 種類 | 前置詞 |
|------|--------|
| 位置・場所 | in / on / at / under / between / among / behind |
| 時 | at / on / in / before / after / during / since / for |
| 方向 | to / from / into / out of / toward / through |
| 関係 | with / by / for / about / against |

### 前置詞句は Y か Z か

**文の中での位置**で決まります（形ではなく位置と働き → 第14章）。

- **名詞の直後** → Y（その名詞を後置修飾）  
  the book **on the desk** → "on the desk" が "book" を修飾
- **動詞の周辺** → Z（その動詞を修飾）  
  I study **in the library** → "in the library" が "study" を修飾

**同じ形、違う役割**

| 文 | 前置詞句 | 役割 |
|----|---------|------|
| the book on the desk | on the desk | Y（book を修飾） |
| I read in the library | in the library | Z（read を修飾） |

### 副詞として機能する単語との対比

`home / here / there / abroad / upstairs / inside / outside / away / back` などは **それ自体が副詞** です。前置詞不要で Z として使えます。

```
○ go home / × go to home
○ come here / × come to here
```

**なぜ × なのか**

`home` や `here` は「場所を表す副詞」として単独で機能するため、`to` などの前置詞は不要です。

ただし所有格を付けて名詞扱いにすれば、前置詞が必要になります。

```
○ go to my home（"home" を名詞として使う）
```

## 第18章　接続詞

> アプリ該当：Step 6 で集中訓練

### この章で扱うこと

**接続詞**は、語や文、節と節をつなぐ機能語です。種類によって、つなぐ要素の関係（対等・従属・名詞節の導入など）が決まります。

### 接続詞は4種類

| 種類 | 役割 | 代表例 |
|------|------|--------|
| 等位接続詞 | 対等な要素をつなぐ | for / and / nor / but / or / yet / so |
| 従属接続詞（副詞節用）| 副詞節を導入 | when / because / if / although / while |
| 名詞節導入語 | 名詞節を導入 | that / whether / if |
| 相関接続詞 | 相関的に対をなす | both...and / either...or / neither...nor / not only...but also |

### 等位接続詞 FANBOYS

等位接続詞は **FANBOYS** の頭文字で覚えられます。

**F**or（理由）/ **A**nd / **N**or / **B**ut / **O**r / **Y**et（しかし）/ **S**o（だから）

同じ品詞・同じ機能のものをつなぎます。

```
I bought bread and butter.（名詞 and 名詞）
She studied hard but failed.（動詞 but 動詞）
```

**読み方**

- `and` → 追加・並列
- `but` → 対比・逆接
- `so` → 結果

### 従属接続詞 + 節 = Z（副詞節）

従属接続詞の後ろに S+V が続くと、**副詞節（Z）** になります。詳細は第12章を参照。

意味カテゴリで MECE に分かれます：時・理由・条件・譲歩・目的・結果・様態・比較・場所。

## 第19章　関係詞

> アプリ該当：Step 5 で集中訓練

### この章で扱うこと

**関係詞**は、節を **Y（後置修飾）** または **X（名詞節）** に変換する機能語です。第2章で学んだ「ここから説明が始まる合図」の正体が、ここにあります。

### 関係詞は2系統

| 種類 | 形 | 役割 |
|------|----|----|
| 関係代名詞 | who / whom / whose / which / that / what | 名詞の代わりに節に入り、節全体を Y にする（what だけ X）|
| 関係副詞 | where / when / why | 副詞の代わりに節に入り、節全体を Y にする |

### 関係代名詞の使い分け

| 先行詞 | 主格 | 目的格 | 所有格 |
|--------|------|--------|--------|
| 人 | who / that | whom / who / that / 省略 | whose |
| 物・事 | which / that | which / that / 省略 | whose / of which |

**主格・目的格の見分け**

- 主格：節内で主語になる → `a man **who** speaks English`
- 目的格：節内で目的語になる → `the book (that) I bought`（I bought the book）

### 関係副詞の使い分け

| 先行詞 | 関係副詞 | 例 |
|--------|---------|----|
| 場所 | where | the city **where I live** |
| 時 | when | the day **when we met** |
| 理由 | why | the reason **why he left** |

### 関係代名詞 what の特別性

`what` は **先行詞を含む** 関係代名詞です。「〜するもの・こと」の意味になります。

```
What he said is true.   = The thing that he said is true.
                          ↑ X（名詞節）
```

→ what 節は Y ではなく **X（名詞節）** として機能します。`what` の前に名詞を置けません（× I have a pen what you borrowed）。

### 関係詞と疑問文の連続性

関係詞節と疑問文は、同じ **「空所を埋める」** 発想で作られます。

| 用途 | 空所の埋め方 |
|------|------------|
| 疑問文 | 空所を **文頭に出して「聞く」** |
| 関係詞節 | 空所を **文頭に出して「名詞を説明する」** |

```
疑問文：       What did you buy?         （空所＝何）
関係詞節：    the thing that you bought  （空所＝何）
```

疑問文の `What` が文頭に来るのと、関係詞節の `that` が名詞の後ろに来るのは、**空所の扱い方の違い**です。

## 第20章　句動詞

> アプリ該当：Step 4 以降で随時

### この章で扱うこと

**句動詞**は、複数の語が一体となって **1つの動詞** として機能するものです。V（動詞）のかたまりは、必ずしも1語とは限りません。

### 句動詞とは

複数の語が一体となって1つの動詞として機能するもの。V のかたまりは1語とは限りません。

**なぜ大事か**

`look at` を「look（見る）+ at（前置詞）」と別々に読むと、骨格が崩れます。**look at 全体が1つのV** として扱う必要があります。

### 種類

| 種類 | 例 |
|------|-----|
| 動詞 + 前置詞 | look at / listen to / depend on |
| 動詞 + 副詞（自動詞的）| give up / break down / show up |
| 動詞 + 副詞 + 目的語 | turn on the light / pick up the book |
| 動詞 + 副詞 + 前置詞 | look up to / put up with / get away with |

### 読解上の注意

句動詞を「動詞 + 別物」と読むと骨格を見失います。**動詞 + α 全体で1つのV** として扱いましょう。

```
She looked at the picture.
    └────V────┘  └──X(目的語)──┘
```

- V = `looked at`（見た）
- X = `the picture`（目的語）

`at` を前置詞として単独に切り離さず、`looked at` をセットで V と見なします。

## 第21章　名詞句の内部構造

> アプリ該当：第13章の補足

### この章で扱うこと

名詞句（X として機能する Phrase）の**中身**も、X / Y / Z で分解できます。ネスト（入れ子）を最も身近に観察できる場所です。

### 名詞句の中もまた X / Y / Z

X（名詞役）の Phrase = 名詞句 は、内部にも構造を持ちます。

```
[限定詞] [数詞] [前置形容詞] 名詞 [後置修飾句・節]
   the      two       tall      men    in the park / who came
                                       ↑ Y/Phrase / Y/Clause
```

### 内部の役割

| 部品 | 役割 |
|------|------|
| the / two | 機能語（限定詞・数詞）|
| tall | Y/Word（前置形容詞）|
| men | X/Word（名詞の核）|
| in the park | Y/Phrase（後置修飾）|
| who came | Y/Clause（後置修飾）|

**読み方**

1. 名詞句全体 = X（主語・目的語など）
2. 中の `men` = 名詞の核（X/Word）
3. `tall` = 名詞を修飾（Y/Word・前置）
4. `in the park` / `who came` = 名詞を修飾（Y/Phrase・Y/Clause・後置）

→ X/Y/Z の再帰性を **最も身近に観察できる場所** です。

## 第22章　ニュアンスと表現の選択

> アプリ該当：全Stepで意識

### この章で扱うこと

同じ日本語でも、英語では**複数の構造**で訳せることがあります。ニュアンスの違いを意識して、適切な構造を選ぶことが大切です。

### 同じ日本語でも英語の構造は1対1でない

```
「彼女には読むべき本がたくさんある」
```

これは少なくとも以下3通りに訳せます。

| 英訳 | ニュアンス |
|------|-----------|
| She has many books **to read**. | 積読・読む予定（量の話）|
| She has many books **she should read**. | 外部起点の推奨・義務 |
| She has many books **worth reading**. | 読む価値がある（価値判断）|

**ポイント**

- 日本語1文 → 英語1文、とは限らない
- 同じ「読むべき本」でも、**義務・予定・価値** などニュアンスで構造が変わる

### 不定詞（to do）の Y 用法のニュアンス

不定詞の Y 用法（名詞の後置修飾）は、**「これから／予定／可能」** を表します。義務や推奨ではありません。

```
a book to read       = これから読む本
something to eat     = 食べるもの（これから食べる）
a place to visit     = 訪れる場所（予定）
```

### 義務・推奨を表したいとき

| 表現 | 構造 |
|------|------|
| books she should read | 関係詞節（Y/Clause）|
| books worth reading | 形容詞句（Y/Phrase）|
| recommended books | 過去分詞（Y/Word）|

### 学習者の意識

英作文では、**「自分の日本語のニュアンスがどれか」** を見極めてから構造を選びましょう。同じ日本語に複数の英訳がありうることを前提に置くことが、正確な表現につながります。

## 第23章　その他の構文

> アプリ該当：Step 7（将来）

### この章で扱うこと

本書の主軸（骨格・X/Y/Z・ネスト）以外に、Step 7 以降で扱う**発展的な構文**を一覧します。現時点では後回しにして構いません。

### 比較

| 種類 | 形 | 例 |
|------|----|----|
| 原級 | as ... as | as tall as me |
| 比較級 | -er than / more ... than | taller than me |
| 最上級 | the -est / the most ... | the tallest |

### 仮定法

| 種類 | 形 | 例 |
|------|----|----|
| 仮定法過去（現在の反実）| If S 過去形, S would 原形 | If I were you, I would go. |
| 仮定法過去完了（過去の反実）| If S had + pp, S would have + pp | If I had known, I would have helped. |
| 仮定法未来 | If S should / were to ... | If it should rain, ... |

### 倒置・強調

| 種類 | 形 | 例 |
|------|----|----|
| 倒置 | 否定語 + 助動詞 + S + V | **Never have I seen** such a thing. |
| it-cleft | It is X that ... | **It is John** that I met. |
| what-cleft | What S V is ... | **What I want** is rest. |

### 特殊構文

| 種類 | 形 | 例 |
|------|----|----|
| there構文 | There + be + 名詞 | There is a book on the desk. |
| 命令文 | 動詞原形から始める | Open the door. |
| 感嘆文 | How / What | How beautiful! / What a day! |

## 付録 B. 文を解く順序（実践パーシング）

長い英文に遭遇したとき、**どの順番で分解するか** の実践手順です。第1章〜第15章の内容を、実際の文に当てはめるためのチェックリストとして使ってください。

### Step 1：骨格を見つける

1. **動詞（V）を見つける**：時制が付いている動詞が骨格の動詞
2. **その左の主語（S）を確定する**：動詞の前にある最も近い名詞要素
3. **動詞のタイプから O / C の有無を予測する**（→ 第8章）

**ポイント**：いきなり修飾語から読み始めない。まず「だれが・何をした」を固定する。

### Step 2：各位置のかたまりを確定する

S・O・C それぞれが、次のどれかを確認します。

- 単語の名詞か？
- 句か（動名詞句・不定詞句・名詞句）？
- 節か（that節・wh節・whether節）？

### Step 3：修飾要素（Y / Z）を切り分ける

骨格に含まれない部分は Y か Z です。

- 名詞の直後にあるものは Y（後置修飾）と仮定
- 動詞の周辺にあるものは Z と仮定
- 文頭・文末の独立した句・節は Z の可能性が高い

### Step 4：かたまりの中にネストがあれば再帰的に同じ手順を適用

節の中身も同じ手順で骨格を見つけます（→ 第15章）。**Step 1〜4 を、節の中で繰り返す**イメージです。

### 例題

```
The student who studies hard usually gets good grades.
```

**Step 1：骨格**

1. 動詞を探す：`studies` と `gets` の2つ。どちらも現在形。
2. 骨格の動詞は文末側の `gets`。`studies` は内部の節（who節）の動詞。
3. 主語：`The student` ＋ 後ろの関係詞節 = X

**Step 2：O / C**

- 動詞 `gets` の後ろ：`good grades` = X（目的語）

**Step 3：修飾**

- `who studies hard` = Y（関係詞節、student を修飾）
- `usually` = Z（中置・頻度副詞）

**最終構造**

```
[The student who studies hard] usually gets [good grades].
        X（主語・名詞句 + Y）       Z       V       X（目的語）
```
